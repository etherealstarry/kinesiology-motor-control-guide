    实验设备使用

    !!! info "设备培训的重要性" TAMU KNSM 实验室的设备需要在正式使用前完成安全培训和设备认证。未经认证不得使用！

    🔬 常见实验设备

    1️⃣ 运动捕捉系统 (Motion Capture)

    Vicon 系统

    TAMU KNSM 配备：Vicon 红外摄像头系统（通常 8-16 个摄像头）

    主要用途： - 三维运动学数据采集 - 步态分析 - 到达运动 (reaching) 轨迹记录 - 关节角度计算

    基本操作流程：

    1. 开启摄像头 + 控制器
2. 在 Nexus 软件中新建 Trial
3. 校准摄像头（Wand Wave）
4. 贴标记点（Marker Placement）
5. 静态校准（Static Trial）→ 建立骨骼模型
6. 动态采集（Dynamic Trial）→ 记录运动
7. 后处理：标签修复、滤波、导出数据

    培训要求： - 完成 Vicon 官方培训视频 - 通过实验室内部认证 - 首次使用需在实验室管理员监督下进行

    推荐学习资源： - 📖 Vicon Nexus 用户手册 - 🎥 Vicon YouTube 教程 - 📖 Baker, R. Gait Analysis Methods in Rehabilitation. J Neuroeng Rehabil.

    OptiTrack 系统

    特点：性价比高，适合高密度标记点实验

    基本操作（与 Vicon 类似）： - Motive 软件 - 校准 wand → 标定空间 - 贴标记点 → 记录 Trial

    2️⃣ 测力台 (Force Plates)

    TAMU KNSM 常见型号：AMTI 或 Bertec 测力台

    主要用途： - 地面反作用力 (GRF) 测量 - 步态中重心转移分析 - 平衡能力评估 - 运动控制实验中的力学数据

    注意事项： !!! warning "重要安全提醒" - 测力台非常昂贵且易损（一台 $10,000+） - 不要超过最大承重（通常 2,000 N） - 不要踩在测力台边缘 - 每次实验前做调零 (Zeroing) 操作 - 定期做静态校准

    数据处理： - 用 MATLAB 或 Python 读取 .asc / .c3d 文件 - 滤波（4th-order Butterworth, 低通 50-100 Hz） - 计算 COP (Center of Pressure)、GRF 峰值等

    3️⃣ 表面肌电图 (sEMG)

    主要用途： - 肌肉激活时序分析 - 肌肉协同 (Muscle Synergies) 分析 - 疲劳评估

    设备品牌： - Delsys（无线，常用） - Noraxon - TMSi

    操作流程：

    1. 皮肤准备（刮毛 → 打磨 → 酒精擦拭）
2. 放置电极（沿肌纤维方向）
3. 检查阻抗（通常 < 10 kΩ）
4. 开始采集
5. 后处理：整流 (Rectification) → 滤波 → 归一化

    注意事项： !!! tip "EMG 最佳实践" - 电极间距通常为 20-30 mm - 参考电极放在骨性标志上（如尺骨茎突） - 肌电信号容易受50/60 Hz 电源干扰 → 用 Notch 滤波 - 归一化：用 MVC (Maximal Voluntary Contraction) 或 %MVC

    推荐教材： - 📖 De Luca, C. J. Electromyography: Physiology, Engineering, and Noninvasive Applications. Wiley.

    4️⃣ 经颅磁刺激 (TMS)

    !!! danger "高风险设备！需要专门认证" TMS 设备会产生强磁场，有以下禁忌症的人员不能使用： - 有金属植入物（尤其是头部） - 癫痫病史 - 怀孕（谨慎）

    使用 TMS 前必须完成 **CITI 人类被试认证** + **实验室 TMS 安全认证**。

    主要用途： - 运动皮层兴奋性评估（MEP 振幅） - 因果性干预运动学习 - 连接性研究（运动皮层 → 脊髓）

    TAMU 相关实验室：咨询你的导师，是否有 TMS 设备可用。

    推荐学习资源： - 📖 Rossini, P. M. et al. Non-invasive electrical and magnetic stimulation of the brain. Clin Neurophysiol. - 🎥 TMS 安全指南 (ISC)

    5️⃣ 功能性近红外光谱 (fNIRS)

    主要用途： - 运动任务中皮层血氧响应 - 运动学习过程中的皮层可塑性 - 相比 fMRI，允许自由运动

    设备品牌： - NIRx - Hitachi ETG-4000

    注意事项： - 头发茂密区域信号质量差 → 选择前额叶或运动皮层区域 - 运动伪影（Motion Artifact）是主要噪声来源 - 空间分辨率低（相比 fMRI）

    6️⃣ 眼动仪 (Eye Tracker)

    主要用途： - 运动规划中的眼球运动（扫视、平滑追踪） - 视觉注意与运动控制的关系 - 运动学习中的注视模式变化

    设备品牌： - EyeLink（高采样率，研究标准） - Tobii（便携，适合行为实验）

    7️⃣ 脑电图 (EEG)

    主要用途： - 运动准备电位 (Bereitschaftspotential, BP) - 运动相关皮层电位 (MRCP) - 感觉运动节律 (SMR, 8-12 Hz) 事件相关去同步 (ERD/ERS) - 错误相关负波 (ERN/Ne)

    设备品牌： - BrainProducts (TAMU 常用，ActiCap) - BioSemi - EGI (Electrical Geodesics)

    操作流程：

    1. 测量头围 → 选合适帽号
2. 佩戴脑电帽 → 找 Cz、Fz、Cz、Pz 标志点
3. 注入导电膏（每根电极阻抗 < 10 kΩ）
4. 开始采集（BrainVision Recorder / MNE-Python）
5. 后处理：滤波 → 坏导插值 → 重参考 → 分段

    注意事项： !!! tip "EEG 最佳实践" - 采样率至少 1000 Hz（运动研究需要高时间分辨率） - 运动伪影是最大挑战 → 用 ICA 去除眼电/肌电伪影 - 参考电极：Cephalic 参考 或 平均参考 (Average Reference) - 分析工具推荐：MNE-Python（免费、功能强大）

    推荐学习资源：

    资源 类型 说明 链接
    MNE-Python 官方教程 文档+代码 EEG 处理全流程 mne.tools
    EEG 运动控制研究综述 论文 Tanchiatco et al. 2023 PubMed
    Luck: Event-Related Potentials 教材 ERP 分析圣经 搜索 "Luck ERP 4th edition"

    8️⃣ 功能性磁共振成像 (fMRI)

    !!! danger "高风险设备！需要专门认证" fMRI 涉及强磁场（3T），有以下绝对禁忌症的人员不得进入扫描室： - 心脏起搏器、金属植入物（尤其头部/胸部） - 人工耳蜗、胰岛素泵等电子设备 - 纹身墨水含金属（需提前确认） - 怀孕（谨慎，通常允许但需告知）

    使用 fMRI 前必须完成 **CITI 人类被试认证** + **fMRI 安全认证**。

    主要用途： - 运动任务中的脑激活定位（运动皮层、小脑、基底节） - 静息态功能连接（运动网络） - 运动学习过程中的激活变化

    TAMU 相关资源： - TAMU 有 Center for Population Health and Aging (CPHA) 的 fMRI 设备 - 咨询你的导师是否有扫描配额 (Scan Time)

    注意事项： - 运动伪影 (Motion Artifact) 是 fMRI 的最大噪声来源 - 使用 FSL、SPM 或 AFNI 进行数据处理 - 推荐学习：fMRI 数据处理教程 (Duke)

    9️⃣ 经颅电刺激 (tES/tDCS)

    主要用途： - 运动皮层兴奋性调节（阳极刺激 → 兴奋性增加） - 运动学习增强（在线/离线刺激效果） - 感觉运动网络可塑性研究

    设备品牌： - Soterix Medical - NeuroConn

    注意事项： !!! warning "安全规范" - 电流强度通常 1-2 mA（不超过 2 mA） - 刺激时长通常 10-20 分钟 - 电极位置按照 10-20 系统（如：阳极 C3，阴极对侧眶上） - 必须获得 伦理委员会批准 - 对照条件：Sham (假刺激) 必须设置

    💻 数据采集代码模板

    Python 数据采集模板（串口/USB 设备）

    """
TAMU KNSM 实验室数据采集模板
适用于：测力台、EMG、TMS 设备
"""

import serial
import numpy as np
import time
from datetime import datetime

class DataAcquisition:
    def __init__(self, port='/dev/cu.usbmodem', baudrate=115200):
        self.port = port
        self.baudrate = baudrate
        self.ser = None
        self.data = []

    def connect(self):
        """连接设备"""
        try:
            self.ser = serial.Serial(self.port, self.baudrate, timeout=1)
            print(f"✅ 已连接 {self.port}")
        except Exception as e:
            print(f"❌ 连接失败: {e}")

    def start_acquisition(self, duration_sec=60):
        """开始采集数据"""
        print(f"开始采集 {duration_sec} 秒...")
        start_time = time.time()

        while (time.time() - start_time) < duration_sec:
            if self.ser.in_waiting:
                line = self.ser.readline().decode('utf-8').strip()
                timestamp = datetime.now().isoformat()
                self.data.append([timestamp, line])
                print(f"[{timestamp}] {line}")

        print("✅ 采集完成")

    def save_data(self, filename):
        """保存为 CSV"""
        import pandas as pd
        df = pd.DataFrame(self.data, columns=['timestamp', 'value'])
        df.to_csv(filename, index=False)
        print(f"💾 数据已保存: {filename}")

    def disconnect(self):
        """断开连接"""
        if self.ser:
            self.ser.close()
            print("🔌 已断开连接")

# 使用示例
if __name__ == "__main__":
    daq = DataAcquisition(port='/dev/cu.usbmodem', baudrate=115200)
    daq.connect()
    daq.start_acquisition(duration_sec=30)
    daq.save_data(f"data/raw/sub-01_task-motor_run-01.csv")
    daq.disconnect()

    数据处理流程（Python）

    """
运动科学数据处理标准流程
适用：EMG、测力台、运动捕捉数据
"""

import numpy as np
import pandas as pd
from scipy import signal
import matplotlib.pyplot as plt

def process_emg(data, fs=1000, lowpass=20, highpass=450):
    """
    处理 EMG 信号
    - 带通滤波 (20-450 Hz)
    - 整流 (Rectify)
    - 平滑 (Moving Average)
    """
    # 带通滤波
    b, a = signal.butter(4, [lowpass/(fs/2), highpass/(fs/2)], btype='band')
    filtered = signal.filtfilt(b, a, data)

    # 整流
    rectified = np.abs(filtered)

    # 平滑（移动平均，窗口 50 ms）
    window = int(0.05 * fs)
    smoothed = pd.Series(rectified).rolling(window=window, center=True).mean()

    return smoothed.values

def process_force_plate(data, fs=1000):
    """
    处理测力台数据
    - 计算 COP (Center of Pressure)
    - 检测触地/离地时刻
    """
    fx, fy, fz = data[:, 0], data[:, 1], data[:, 2]
    mx, my = data[:, 3], data[:, 4]

    # COP 计算
    cop_x = -my / fz
    cop_y = mx / fz

    # 触地检测（Fz > 20 N 为触地）
    contact = fz > 20

    return cop_x, cop_y, contact

# 使用示例
if __name__ == "__main__":
    # 读取数据
    data = np.loadtxt('data/raw/emg_sub01_run01.csv', delimiter=',')

    # 处理 EMG
    emg_processed = process_emg(data[:, 0])

    # 可视化
    plt.figure(figsize=(10, 4))
    plt.plot(emg_processed)
    plt.xlabel('采样点'); plt.ylabel('EMG (mV)')
    plt.title('处理后 EMG 信号')
    plt.savefig('data/derived/emg_sub01_processed.png', dpi=300)
    plt.show()

    🛡️ 设备使用安全规范

    通用安全原则

    1. 培训第一：未经培训不得擅自操作设备
    2. 记录使用日志：每次使用记录在实验室 Logbook
    3. 报告故障：设备异常立即报告实验室管理员
    4. 清洁消毒：与人体接触的设备每次使用后清洁（尤其 EEG/EMG 电极）

    数据存储规范

    !!! success "数据管理最佳实践" ``` 推荐目录结构： data/ ├── raw/ # 原始数据（只读，不修改） │ ├── sub-01/ │ ├── sub-02/ │ └── ... ├── derived/ # 处理后的数据 │ ├── filtered/ │ └── features/ ├── behavioral/ # 行为数据 └── logs/ # 实验日志

    备份策略：
├── 本地硬盘（工作副本）
├── 实验室服务器（每周同步）
└── 云端备份（Box/OneDrive，每月同步）

    📋 设备预约系统

    TAMU KNSM 通常使用在线预约系统（如 Google Calendar 或专用软件）：

    • 提前预约：至少提前 24 小时
    • 准时开始和结束：尊重下一位使用者的时间
    • 取消政策：无法使用时提前取消（至少提前 4 小时）
    • 设备培训预约：联系实验室管理员安排
