    数据分析工具

    !!! info "工具选择指南" 运动神经科学研究需要多种数据分析工具。根据你的研究方向、编程基础和长期职业规划选择合适工具。

    📊 主流数据分析工具对比

    工具 学习曲线 优势 劣势 推荐场景
    R 中等 统计包最全、ggplot2 可视化强 编程门槛 ⭐⭐⭐⭐⭐ 最推荐
    Python 中等 通用性强、机器学习生态好 统计细节不如 R ⭐⭐⭐⭐ 强烈推荐
    MATLAB 较陡 信号处理工具箱强 昂贵、闭源 ⭐⭐⭐ 神经信号分析
    JASP 平缓 点按操作、贝叶斯统计 灵活性有限 ⭐⭐⭐ 快速探索
    SPSS 平缓 广泛接受、操作简单 灵活性差、昂贵 ⭐⭐ 不推荐新生使用
    Julia 较陡 速度快、科学计算强 生态不成熟 ⭐⭐ 进阶用户

    R：统计分析与可视化首选

    为什么选 R？

    • ✅ 统计分析包最丰富：ANOVA、混合效应模型、IRT、SEM 全覆盖
    • ✅ ggplot2：科学可视化标杆，发表论文级图表
    • ✅ 开源免费：不用担心许可证问题
    • ✅ RMarkdown/Quarto：可重复性报告，分析 + 文档合一
    • ✅ 学术界标准：绝大多数统计方法论文都提供 R 实现

    必学 R 包

    !!! success "运动神经科学常用 R 包" ```r # 数据清洗与操作 library(tidyverse) # dplyr + ggplot2 + 其他 library(tidyr) # 数据整形 library(lubridate) # 时间数据处理

    # 统计分析
library(afex)         # ANOVA（自动计算 Type II/III SS）
library(emmeans)       # 事后检验与简单效应分析
library(lme4)         # 混合效应模型（重复测量核心！）
library(nlme)         # 另一种混合效应模型实现
library(easystats)     # 统一统计结果报告

# 可视化
library(ggplot2)       # 图表绘制
library(patchwork)     # 多图拼接
library(see)           # 美化 ggplot 输出

# 可重复性
library(renv)         # 包版本管理
library(targets)       # 数据分析流水线
```

    学习路径

    Week 1-2:   R 基础（数据结构、子集提取、函数）
Week 3-4:   dplyr 数据操作（filter, select, mutate, group_by）
Week 5-6:   ggplot2 可视化（ scatter, boxplot, line plot）
Week 7-8:   afex + emmeans 做 ANOVA
Week 9-10:  lme4 混合效应模型（重复测量数据核心）
Week 11-12: RMarkdown / Quarto 可重复报告

    推荐学习资源

    • 📖 Wickham, H. & Grolemund, G. R for Data Science (2e). O'Reilly. 免费在线
    • 📖 Winter, B. Statistik: Eine Einführung mit R. Springer.
    • 🎥 Coursera: Data Science Specialization (JHU)
    • 🎥 统计之都: R 语言教程

    Python：通用编程与机器学习

    为什么同时学 Python？

    • 如果你做计算建模（最优反馈控制、强化学习模拟），Python 比 R 更合适
    • 机器学习 / 深度学习生态（scikit-learn, PyTorch, TensorFlow）
    • 神经影像数据处理（nibabel, nilearn, MNE-Python for EEG/MEG）

    必学 Python 包

    !!! info "运动神经科学常用 Python 包" ```python # 数据分析 import pandas as pd # 数据操作（类似 R 的 dplyr） import numpy as np # 数值计算 import scipy.stats as stats # 统计检验

    # 可视化
import matplotlib.pyplot as plt
import seaborn as sns    # 统计图（类似 ggplot2）
import plotly as px      # 交互式图表

# 统计建模
import statsmodels.api as sm  # 统计模型（ANOVA, 回归）
import pingouin as pg    # 简洁的统计分析接口（推荐！）

# 机器学习
from sklearn import linear_model, ensemble, model_selection

# 神经数据
import mne              # EEG/MEG 数据处理
import nilearn          # fMRI 数据处理
```

    推荐学习资源

    • 📖 McKinney, W. Python for Data Analysis. O'Reilly. 免费在线
    • 📖 VanderPlas, J. Python Data Science Handbook. O'Reilly. 免费在线
    • 🎥 Coursera: Python for Data Science (U. of Michigan)

    MATLAB：神经信号分析传统工具

    适用场景

    • 处理高密度 EEG / ECoG 数据（EEGLAB, FieldTrip 工具箱）
    • 神经信号时频分析
    • 许多经典神经科学论文的代码是用 MATLAB 写的

    为什么不推荐作为首选？

    • ❌ 昂贵：学校许可证不一定覆盖你毕业后使用
    • ❌ 闭源：无法查看算法实现细节
    • ❌ 生态萎缩：学术界正逐渐转向 Python

    如果必须用 MATLAB

    重点关注这些工具箱： - EEGLAB：EEG 数据处理（sccn.ucsd.edu/eeglab/） - FieldTrip：时频分析（fieldtriptoolbox.org） - SPM：fMRI 数据分析（fil.ion.ucl.ac.uk/spm/）

    JASP：快速探索与贝叶斯统计

    为什么推荐 JASP？

    • ✅ 完全免费、开源
    • ✅ 点按操作：不需要编程
    • ✅ 贝叶斯统计：支持贝叶斯 ANOVA、t 检验（这是 JASP 的独门优势！）
    • ✅ 输出美观：直接导出论文级结果表格

    适用场景

    • 快速探索数据（不需要写代码）
    • 教授贝叶斯统计课程
    • 给合作者展示分析结果（他们可能不懂 R/Python）

    下载

    • 官网：jasp-stats.org
    • 完全免费，支持 Windows / macOS / Linux

    辅助工具：Python
