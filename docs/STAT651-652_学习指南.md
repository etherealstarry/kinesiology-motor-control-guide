## 🎓 STAT 651: Statistics in Research I

### 课程性质

!!! info "官方描述"
    For graduate students in other disciplines; **non-calculus exposition** of the concepts, methods and usage of statistical data analysis; T-tests, analysis of variance and linear regression.
    
    — 面向非统计专业研究生，**不需要微积分**，从概念到应用逐步推进。

**核心目标**：让你从"会用 SPSS 跑 t-test"变成"理解什么时候该用什么统计方法，以及为什么"。

### 📖 教材

| 项目 | 详情 |
|------|------|
| 教材 | **An Introduction to Statistical Methods and Data Analysis** (7th Edition) |
| 作者 | R. Lyman Ott & Michael Longnecker |
| 出版社 | Cengage |
| ISBN | 978-0357670620 |
| 配套 | 官方有 PDF 版本 + WebAssign 在线习题系统 |

> 这本教材是整个 STAT 651/652 序列的标准教材，可以一本用两学期。

- **WebAssign**：在线作业和考试平台。所有作业和考试都通过 WebAssign 提交。
- **doStat**：TAMU 统计系的课程管理系统，存放 lecture notes、视频和补充材料。

!!! tip "关于 JMP vs R"
    STAT 651 历史上用 JMP，但近年有些 section 开始引入 R。提前学一下 R 基础对后续科研也有用。参考 [Swirl - Learn R in R](https://swirlstats.com/)。

### 📑 完整学习计划（按主题拆解）

#### 模块一：实验设计基础（约 4 周）

| 主题 | 内容 | 为什么重要 | 教材章节 |
|------|------|-----------|----------|
| 实验设计原则 | 随机化、重复、区组、对照 | 所有实验的基础逻辑 | Ott Ch.7-8 |
| **完全随机设计 (CRD)** | 随机分配、处理效应、单因素方差分析 | 最简单但最常用的设计 | Ott Ch.8 |
| 多重比较 | Tukey HSD, Bonferroni, 事前/事后比较 | 找差异但不能乱找 | Ott Ch.9 |
| 统计功效与样本量 | Cohen's d, Type II error, Power curves | 你的实验需要多少被试？ | Ott Ch.5 |

#### 模块二：区组与因子设计（约 4 周）

| 主题 | 内容 | 应用场景 |
|------|------|----------|
| **随机区组设计 (RCBD)** | 区组因素的识别、区组效应 | 控制个体差异（如：不同被试基线不同） |
| **因子设计 (Factorial)** | 主效应、交互效应、2^k 设计 | 同时研究训练方式 × 强度的交互作用 |
| **裂区设计 (Split-Plot)** | 混合效应模型、nesting | 一个因素难随机化时（如：性别+训练方案） |
| **重复测量设计** | Within-subject 因素、球对称性假设 | 运动学习实验标配（前测-后测-保留测） |

#### 模块三：多元回归（约 4 周）

| 主题 | 内容 | 考试重点 |
|------|------|----------|
| 多元线性回归 | 多个预测变量、偏回归系数 | β 系数的解释 |
| 模型选择 | 逐步回归、AIC/BIC、R² adj | 哪个模型更好？ |
| 交互项与多项式 | 调节效应、非线性关系 | 交互项怎么解读 |
| 回归诊断 | 共线性 VIF、残差图、Cook's D | 模型假设检查 |

#### 模块四：高级方法（约 2 周）

| 主题 | 内容 |
|------|------|
| **ANCOVA** | 协方差分析——在 ANOVA 中控制连续型协变量（如：控制年龄后再比较训练效果） |
| **非参数方法** | Friedman 检验（重复测量的非参数版）、Wilcoxon 符号秩 |
| **卡方检验进阶** | 分层卡方、Mantel-Haenszel |
| **样本调查方法** | 分层抽样、整群抽样、调查权重 |

### 🔥 学生生存指南

根据历年学生的经验和教训，以下是在 Akleman 的 STAT 652 拿到 A 的关键策略：

#### Rule 1: 教材是命根子

!!! danger "最重要的一条"
    多名学生一致指出：**PPT 不够用**。她的 PPT 是摘要式提纲，考试考的是教材里的细节和例题。每节课后必须读对应章节的教材。

    推荐方法：
    - 课前 30 分钟：看 PPT 了解本章框架
    - 课后 2 小时：精读教材对应章节，做章末奇数题（答案在教材后）
    - 考前：章末偶数题 + Student Solutions Manual 全部过一遍

#### Rule 2: WebAssign 作业拿满分

> 作业占 25%，有 2 次提交机会。这是你最能控制的部分。策略：
> - 第一次提交：独立完成，不确定的标记出来
> - 对照教材和笔记检查 → 第二次提交修正

#### Rule 3: 考试时间管理

> 多名学生反映考试**时间很紧**：
> - "Watch lectures and go over the book about two times before the exam so you can finish on time."
> - 提前做一张 **A4 纸公式 + 方法选择流程图**（open book 但没时间翻书）

#### Rule 4: 不要指望 Curve

> 2016 年学生明确说 "NO curves"。做好没 curve 的准备，每分都靠自己挣。

#### Rule 5: 教材版本

教材与 STAT 651 相同：**Ott & Longnecker, An Introduction to Statistical Methods and Data Analysis**（第 7 版）。一本用两学期。强烈建议买 Student Solutions Manual。

#### Rule 6: 利用她的 Office Hour

> 虽然她的回答风格可能简练，但她是**有问必答**的。带着具体问题去 OH，比如："我在做 Chapter 12 的 Exercise 5 时不太理解为什么用 Latin Square design 而不是 RCBD，您能帮我分析一下吗？"——而不是泛泛地说"我不懂这章"。

## 🔬 考题分析

### STAT 651 考试结构和规律

!!! warning "基于历年 syllabus 的模式分析"

**Exam 1（第 12 讲，覆盖第 1-11 讲）**

| 题型 | 占比 | 说明 |
|------|------|------|
| 选择题 | ~40% | 概念理解（如：什么时候用 t 检验 vs ANOVA） |
| 计算题 | ~40% | 给数据，选方法，算结果 |
| 解释题 | ~20% | 给 SPSS/JMP 输出，解释含义 |

**重点考点**：
1. 描述统计 → 推断统计的过渡（为什么需要推断？）
2. t 检验 vs Wilcoxon 的选择条件（正态性假设）
3. ANOVA 的 F 统计量含义（组间变异 / 组内变异）
4. p 值的正确理解（不是"零假设为真的概率"）
5. Type I vs Type II 错误的应用场景

**Exam 2（第 20 讲，覆盖第 1-19 讲，重点在 12-19）**

**重点考点**：
1. ANOVA 的假设检验（正态性、方差齐性、独立性）
2. 事后比较（为什么不能直接做多次 t 检验）
3. 卡方检验的适用条件（期望频数 ≥ 5）
4. 非参数方法的适用场景

**Final Exam（覆盖全部 28 讲）**

!!! danger "Final 是最容易翻车的"
    **重点考点**：
    1. 回归分析（斜率解释、R²、残差诊断）— 通常占 Finals 的 40%+
    2. 相关性 vs 因果性 — 必考概念题
    3. ANCOVA 的基本逻辑 — 为什么要加协变量
    4. 全学期内容的综合应用题

### STAT 651 考试特点总结

1. **Open book / open notes**：考试可以带任何材料，但时间紧张（2小时完成所有题目），所以**提前准备公式总结表是关键**。
2. **WebAssign 在线考试**：3 次提交机会，最后一次的答案算分，答案不会告诉你对错（所以第一次提交后没法改）。
3. **不会 Curve**：89.45 = A 是不变的。靠摸鱼划水不可能混到 A。

## 🔥 五步复习法（适配统计课）

!!! abstract "基于 KINE 406/426 学习指南中的五步复习法，适配统计课程"

### Step 1: 概念地图 (Concept Map) — 课前 30 分钟

```
本周主题（如：ANOVA）

├── 我需要什么前提知识？
│   ├── 正态分布假设 → 检验方法：Shapiro-Wilk
│   ├── 方差齐性假设 → 检验方法：Levene's Test
│   └── 独立性假设 → 实验设计保证
│
├── 核心问题是什么？
│   └── "三组以上的均值比较，为什么不能做多次 t 检验？"
│       └── 答案：多重比较会膨胀 Type I 错误率
│
├── 主要方法有哪些？
│   ├── One-way ANOVA (单因素)
│   ├── Two-way ANOVA (双因素 → 交互效应)
│   └── Repeated Measures ANOVA (重复测量)
│
└── 结果怎么报告？
    └── F(df1, df2) = X.XX, p = .XXX, η² = .XX
```

### Step 2: 刷教材例题 — 课后 1 小时

1. **每讲后做 3-5 道教材例题**（不做课后习题，先做题内的 Solved Example）
2. 先自己算 → 再对答案 → 标记卡住的地方
3. 用 JMP/SPSS 重复算一遍（手算 + 软件双重验证）

### Step 3: 做概念卡片 — 每周 2 次，每次 20 分钟

| 正面（问题） | 背面（答案） |
|-------------|-------------|
| "t 检验的假设有哪些？" | 正态性 + 独立性 + （等方差） |
| "p < 0.05 意味着什么？" | 如果零假设为真，观察到这样的数据（或更极端）的概率 < 5% |
| "什么时候用 Wilcoxon 代替 t？" | 数据严重非正态或有离群值 |
| "R² = 0.75 怎么解释？" | 模型解释了 75% 的因变量变异 |
| "ANOVA 中 F 值很大说明什么？" | 组间变异远大于组内变异，至少有一组均值不同 |

- **考前三件事**：
  1. 把教材章末习题全部重做一遍

## 📝 JMP 输出解读速查表

!!! tip "考试中最常见的题型就是给 JMP 输出让你解释"

### t 检验输出 (Oneway Analysis → t Test)

```
                    t Test
Difference           2.45
Std Err Dif          0.82
Upper CL Dif         4.12
Lower CL Dif         0.78
Confidence           0.95
t Ratio              2.99
DF                   28
Prob > |t|           0.006*
```
→ 结论：两组差异显著，t(28) = 2.99, p = 0.006

### ANOVA 输出 (Fit Y by X → Oneway)

```
Source      DF    Sum of Squares    Mean Square    F Ratio    Prob > F
Group        2         156.30          78.15        5.42       0.011*
Error       27         389.12          14.41
Total       29         545.42
```
→ 结论：至少有一组均值不同，F(2, 27) = 5.42, p = 0.011

### 回归输出 (Fit Y by X → Fit Line)

```
Summary of Fit
RSquare             0.723
RSquare Adj         0.711

Parameter Estimates
Term        Estimate    Std Error    t Ratio    Prob > |t|
Intercept     12.34        2.15        5.74       <.0001*
X              3.56        0.42        8.48       <.0001*
```
→ 结论：X 每增加 1 单位，Y 平均增加 3.56 单位，模型解释了 72.3% 的变异

## 📊 STAT 651 自测模拟题

### 一、概念题（理解检验）

**Q1**：一位研究者想比较三种训练方法（A, B, C）对反应时间的影响。为什么不能做三次 t 检验（A vs B, A vs C, B vs C）？

**期望答案**：每做一次 t 检验，Type I 错误概率 α = 0.05。三次独立的 t 检验，至少犯一次错误的概率 = 1 - (0.95)³ ≈ 0.143，远超 0.05。ANOVA 一次性控制整体错误率。

**Q2**：p = 0.04。这个结果意味着"零假设为真的概率是 4%"吗？

**期望答案**：❌ 不是。p 值不是零假设为真的概率。p = 0.04 意味着：**如果**零假设为真，观察到这样的数据（或更极端）的概率是 4%。

**Q3**：什么是 Type II 错误？小样本量对它的影响是什么？

**期望答案**：Type II 错误 = 零假设实际为假但未拒绝。小样本量 → 统计功效低 → Type II 错误概率增大 → 可能漏掉真实效应。

### 二、计算题

**Q4**：某实验比较控制组 (n=10, Mean=45, SD=8) 和实验组 (n=10, Mean=53, SD=7)。进行独立样本 t 检验。

- 计算 pooled variance
- t 值约等于？
- df = ?

**参考答案**：
- Pooled variance = [(9×64) + (9×49)] / 18 = (576 + 441) / 18 = 56.5
- SE = √(56.5 × (1/10 + 1/10)) = √(11.3) ≈ 3.36
- t = (53 - 45) / 3.36 ≈ 2.38
- df = 18
- 查表：t(0.05, 18) = 2.101，所以 p < 0.05，拒绝零假设

**Q5**：某研究的 ANOVA 输出显示 SS_between = 120, SS_within = 200, df_between = 2, df_within = 27。

- MS_between = ?
- MS_within = ?
- F = ?
- η² = ?

**参考答案**：
- MS_between = 120 / 2 = 60
- MS_within = 200 / 27 ≈ 7.41
- F = 60 / 7.41 ≈ 8.10
- η² = 120 / (120 + 200) = 0.375（37.5% 的变异由分组解释）

### 三、JMP 输出解读

**Q6**：
```
Term        Estimate    Std Error    t Ratio    Prob > |t|
Intercept     25.0         3.2         7.81       <.0001
Hours          3.5         0.45        7.78       <.0001
```
- 写出回归方程
- 如果一个人复习 8 小时，预测成绩是多少？
- 这个模型显著的证据是什么？

**参考答案**：
- 回归方程：成绩 = 25.0 + 3.5 × Hours
- 复习 8 小时 → 成绩 = 25.0 + 3.5 × 8 = 53.0
- Hours 的 p < 0.0001，说明复习时间对成绩有显著正向预测作用
