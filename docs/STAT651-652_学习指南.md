## 🎓 STAT 651: Statistics in Research I

### 课程性质

!!! info "官方描述"
    For graduate students in other disciplines; **non-calculus exposition** of the concepts, methods and usage of statistical data analysis; T-tests, analysis of variance and linear regression.
    
    — 面向非统计专业研究生，**不需要微积分**，从概念到应用逐步推进。

**核心目标**：让你从"会用 SPSS 跑 t-test"变成"理解什么时候该用什么统计方法，以及为什么"。

## 🎓 STAT 652: Statistics in Research II

### 课程性质

!!! info "官方描述"
    Continuation of STAT 651. Concepts of experimental design, individual treatment comparisons, randomized blocks and factorial experiments, **multiple regression**, Chi-squared tests and a brief introduction to covariance, non-parametric methods and sample surveys.

STAT 652 = STAT 651 的**实验设计 + 高级方法**扩展。

**核心目标**：从"会跑统计检验"进阶到"会设计实验 + 选择正确的分析方法"。这是博士生做 dissertation 的核心能力。

与 STAT 651 类似：
- 作业 (WebAssign)：~20%
- Exam 1：~25%
- Exam 2：~25%
- Final Exam：~30%

### 📑 预计教学内容

!!! note "以下内容基于课程描述和历史 syllabus 推断，具体以开学后 syllabus 为准"

#### 实验设计 (Experimental Design)

| 主题 | 核心概念 | 为什么重要 |
|------|----------|-----------|
| **完全随机设计 (CRD)** | 随机分配、处理效应 | 最基本实验设计 |
| **随机区组设计 (RCBD)** | 区组因素、减少误差 | 控制个体差异（运动科学大量使用） |
| **因子设计 (Factorial)** | 主效应、交互效应、2^k 设计 | 同时研究多个因素（如：训练方式 × 强度） |
| **重复测量设计** | Within-subject 因素、球对称性 | 运动学习实验的标配 |
| **裂区设计 (Split-Plot)** | 混合效应、nesting | 复杂的运动科学实验 |

#### 高级方法

| 主题 | 内容 |
|------|------|
| **多元回归 (Multiple Regression)** | 多个预测变量、模型选择、交互项 |
| **ANCOVA** | 协方差分析，控制混淆变量 |
| **非参数方法** | Friedman 检验、Wilcoxon 符号秩 |
| **样本调查 (Sample Surveys)** | 分层抽样、整群抽样 |

!!! warning "基于历年 syllabus 的模式分析"

## 🔥 五步复习法（适配统计课）

!!! abstract "基于 KINE 406/426 学习指南中的五步复习法，适配统计课程"

### Step 1: 概念地图 (Concept Map) — 课前 30 分钟

```
本周主题（如：ANOVA）

├── 我需要什么前提知识？
│   ├── 正态分布假设 → 检验方法：Shapiro-Wilk
│   ├── 方差齐性假设 → 检验方法：Levene's Test
│   └── 独立性假设 → 实验设计保证
│
├── 核心问题是什么？
│   └── "三组以上的均值比较，为什么不能做多次 t 检验？"
│       └── 答案：多重比较会膨胀 Type I 错误率
│
├── 主要方法有哪些？
│   ├── One-way ANOVA (单因素)
│   ├── Two-way ANOVA (双因素 → 交互效应)
│   └── Repeated Measures ANOVA (重复测量)
│
└── 结果怎么报告？
    └── F(df1, df2) = X.XX, p = .XXX, η² = .XX
```

### Step 2: 刷教材例题 — 课后 1 小时

1. **每讲后做 3-5 道教材例题**（不做课后习题，先做题内的 Solved Example）
2. 先自己算 → 再对答案 → 标记卡住的地方
3. 用 JMP/SPSS 重复算一遍（手算 + 软件双重验证）

### Step 3: 做概念卡片 — 每周 2 次，每次 20 分钟

| 正面（问题） | 背面（答案） |
|-------------|-------------|
| "t 检验的假设有哪些？" | 正态性 + 独立性 + （等方差） |
| "p < 0.05 意味着什么？" | 如果零假设为真，观察到这样的数据（或更极端）的概率 < 5% |
| "什么时候用 Wilcoxon 代替 t？" | 数据严重非正态或有离群值 |
| "R² = 0.75 怎么解释？" | 模型解释了 75% 的因变量变异 |
| "ANOVA 中 F 值很大说明什么？" | 组间变异远大于组内变异，至少有一组均值不同 |

```

Difference           2.45

Lower CL Dif         0.78
Confidence           0.95
t Ratio              2.99
DF                   28
Prob > |t|           0.006*
```

→ 结论：两组差异显著，t(28) = 2.99, p = 0.006


→ 结论：至少有一组均值不同，F(2, 27) = 5.42, p = 0.011

## 📊 STAT 651 自测模拟题

### 一、概念题（理解检验）

**Q1**：一位研究者想比较三种训练方法（A, B, C）对反应时间的影响。为什么不能做三次 t 检验（A vs B, A vs C, B vs C）？

**期望答案**：每做一次 t 检验，Type I 错误概率 α = 0.05。三次独立的 t 检验，至少犯一次错误的概率 = 1 - (0.95)³ ≈ 0.143，远超 0.05。ANOVA 一次性控制整体错误率。

**Q2**：p = 0.04。这个结果意味着"零假设为真的概率是 4%"吗？

**期望答案**：❌ 不是。p 值不是零假设为真的概率。p = 0.04 意味着：**如果**零假设为真，观察到这样的数据（或更极端）的概率是 4%。

**Q3**：什么是 Type II 错误？小样本量对它的影响是什么？

**期望答案**：Type II 错误 = 零假设实际为假但未拒绝。小样本量 → 统计功效低 → Type II 错误概率增大 → 可能漏掉真实效应。

### 二、计算题

**Q4**：某实验比较控制组 (n=10, Mean=45, SD=8) 和实验组 (n=10, Mean=53, SD=7)。进行独立样本 t 检验。

- 计算 pooled variance
- t 值约等于？
- df = ?

**参考答案**：
- Pooled variance = [(9×64) + (9×49)] / 18 = (576 + 441) / 18 = 56.5
- SE = √(56.5 × (1/10 + 1/10)) = √(11.3) ≈ 3.36
- t = (53 - 45) / 3.36 ≈ 2.38
- df = 18
- 查表：t(0.05, 18) = 2.101，所以 p < 0.05，拒绝零假设

**Q5**：某研究的 ANOVA 输出显示 SS_between = 120, SS_within = 200, df_between = 2, df_within = 27。

- MS_between = ?
- MS_within = ?
- F = ?
- η² = ?

**参考答案**：
- MS_between = 120 / 2 = 60
- MS_within = 200 / 27 ≈ 7.41
- F = 60 / 7.41 ≈ 8.10
- η² = 120 / (120 + 200) = 0.375（37.5% 的变异由分组解释）

```
Term        Estimate    Std Error    t Ratio    Prob > |t|
Intercept     25.0         3.2         7.81       <.0001
Hours          3.5         0.45        7.78       <.0001
```
- 写出回归方程
- 如果一个人复习 8 小时，预测成绩是多少？
- 这个模型显著的证据是什么？

**参考答案**：
- 回归方程：成绩 = 25.0 + 3.5 × Hours
- 复习 8 小时 → 成绩 = 25.0 + 3.5 × 8 = 53.0
- Hours 的 p < 0.0001，说明复习时间对成绩有显著正向预测作用

## ⚠️ STAT 651/652 对 Kinesiology PhD 的实际意义

### Prelim 统计部分

Prelim Exam 的 Methods 部分通常要求你：
1. 读懂一篇论文的统计方法（什么设计、什么检验、为什么选这个）
2. 批判统计方法的选择（是否适合、有什么局限）
3. 为假设的研究问题选择合适的统计方法

STAT 651/652 学完，这些能力自然就有了。

### Dissertation 研究

博士论文的研究设计、数据分析、结果报告，全程依赖统计。STAT 651/652 提供的是**所有统计方法的基础框架**，后续你用什么软件、做什么分析，都在这个框架上扩展。
