## 🗺️ 如何使用这个指南？

### 如果你是即将入学的直博新生

1. **先读 [使用指南](使用指南.md)**：了解 TAMU 直博的基本情况
2. **再读 [学习规划](学习规划.md)**：了解 4-5 年的整体安排
3. **提前预习核心课程**：KINE 606（Motor Neuroscience I）是最核心的课程，建议提前阅读 *Principles of Neural Science* 和 *Motor Control and Learning*
4. **准备统计基础**：STAT 652/608 对很多人来说比较难，暑假可以提前学 R 或 Python

### 如果你已经在读

1. **按课程查找**：在导航栏找到你当前或即将上的课，查看推荐教材和论文
2. **参考学习规划**：对照自己的进度，看看有没有遗漏的重要技能（如编程、学术写作）
3. **查看研究方向**：了解 Motor Neuroscience 下面的不同研究方向，提前确定自己感兴趣的领域
4. **查看拓展资源**：找到相关学术会议、研究实验室和实习机会

### 如果你在准备资格考试或开题

1. **精读必读书籍**：特别是 *Principles of Neural Science* 相关章节
2. **查看研究方向页面**：了解各个领域的前沿进展
3. **参考学习规划中的资格考试准备建议**
