# 后记

恭喜你！如果你已经读到了这里，说明你已经对运动控制（Motor Control）和运动机能学（Kinesiology）有了相当深入的了解。

## 🎉 写在最后

### 关于这个指南

这个指南的诞生，源于一个简单的想法：

> **"好的学习资源应该被系统地整理和传播。"**

我（EtherealStarry）在学习运动控制的过程中，深感学习资源分散、质量参差不齐的困扰。于是参考了 [PKUFlyingPig](https://github.com/PKUFlyingPig) 同学的 [CS自学指南](https://csdiy.wiki/)，决定为运动控制领域也制作一个类似的学习指南。

### 致谢

在这个指南的制作过程中，我受到了很多启发和帮助：

- **[CS自学指南](https://github.com/PKUFlyingPig/cs-self-learning)**：PKUFlyingPig同学的这个项目给了我巨大的启发，无论是项目结构、内容组织，还是网站风格，都让我受益匪浅。

- **Texas A&M University**：感谢德州农工大学公开的课程设置信息，让这个指南有了系统性的框架。

- **所有开源贡献者**：GitHub上无数开源项目的贡献者，你们的工作让知识传播变得更加容易。

- **运动控制领域的研究者**：这个领域有无数优秀的研究者，你们的论文和著作是这个指南的知识来源。
