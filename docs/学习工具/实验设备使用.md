    TAMU KNSM 实验室的设备需要在正式使用前完成**安全培训**和**设备认证**。未经认证不得使用！

## 🔬 常见实验设备

### 1️⃣ 运动捕捉系统 (Motion Capture)

#### Vicon 系统

**TAMU KNSM 配备**：Vicon 红外摄像头系统（通常 8-16 个摄像头）

**主要用途**：
- 三维运动学数据采集
- 步态分析
- 到达运动 (reaching) 轨迹记录
- 关节角度计算

**基本操作流程**：
```
1. 开启摄像头 + 控制器
2. 在 Nexus 软件中新建 Trial
3. 校准摄像头（Wand Wave）
4. 贴标记点（Marker Placement）
5. 静态校准（Static Trial）→ 建立骨骼模型
6. 动态采集（Dynamic Trial）→ 记录运动
7. 后处理：标签修复、滤波、导出数据
```

**培训要求**：
- 完成 [Vicon 官方培训视频](https://www.vicon.com/support/)
- 通过实验室内部认证
- 首次使用需在实验室管理员监督下进行

**推荐学习资源**：
- 📖 **Vicon Nexus 用户手册**
- 🎥 [Vicon YouTube 教程](https://www.youtube.com/c/ViconMotionSystems)
- 📖 Baker, R. *Gait Analysis Methods in Rehabilitation*. J Neuroeng Rehabil.

---

#### OptiTrack 系统

**特点**：性价比高，适合高密度标记点实验

**基本操作**（与 Vicon 类似）：
- Motive 软件
- 校准 wand → 标定空间
- 贴标记点 → 记录 Trial

### 2️⃣ 测力台 (Force Plates)

**TAMU KNSM 常见型号**：AMTI 或 Bertec 测力台

**主要用途**：
- 地面反作用力 (GRF) 测量
- 步态中重心转移分析
- 平衡能力评估
- 运动控制实验中的力学数据

**注意事项**：
!!! warning "重要安全提醒"
    - 测力台**非常昂贵且易损**（一台 $10,000+）
    - 不要超过最大承重（通常 2,000 N）
    - 不要踩在测力台边缘
    - 每次实验前做**调零 (Zeroing)** 操作
    - 定期做**静态校准**

**数据处理**：
- 用 MATLAB 或 Python 读取 .asc / .c3d 文件
- 滤波（4th-order Butterworth, 低通 50-100 Hz）
- 计算 COP (Center of Pressure)、GRF 峰值等

### 3️⃣ 表面肌电图 (sEMG)

**主要用途**：
- 肌肉激活时序分析
- 肌肉协同 (Muscle Synergies) 分析
- 疲劳评估

**设备品牌**：
- Delsys（无线，常用）
- Noraxon
- TMSi

**操作流程**：
```
1. 皮肤准备（刮毛 → 打磨 → 酒精擦拭）
2. 放置电极（沿肌纤维方向）
3. 检查阻抗（通常 < 10 kΩ）
4. 开始采集
5. 后处理：整流 (Rectification) → 滤波 → 归一化
```

**注意事项**：
!!! tip "EMG 最佳实践"
    - 电极间距通常为 **20-30 mm**
    - 参考电极放在**骨性标志**上（如尺骨茎突）
    - 肌电信号容易受**50/60 Hz 电源干扰** → 用 Notch 滤波
    - 归一化：用 MVC (Maximal Voluntary Contraction) 或 %MVC

**推荐教材**：
- 📖 De Luca, C. J. *Electromyography: Physiology, Engineering, and Noninvasive Applications*. Wiley.

### 4️⃣ 经颅磁刺激 (TMS)

!!! danger "高风险设备！需要专门认证"
    TMS 设备会产生强磁场，有以下**禁忌症**的人员不能使用：
    - 有金属植入物（尤其是头部）
    - 癫痫病史
    - 怀孕（谨慎）
    
    使用 TMS 前必须完成 **CITI 人类被试认证** + **实验室 TMS 安全认证**。

**主要用途**：
- 运动皮层兴奋性评估（MEP 振幅）
- 因果性干预运动学习
- 连接性研究（运动皮层 → 脊髓）

**TAMU 相关实验室**：咨询你的导师，是否有 TMS 设备可用。

**推荐学习资源**：
- 📖 Rossini, P. M. et al. *Non-invasive electrical and magnetic stimulation of the brain*. Clin Neurophysiol.
- 🎥 [TMS 安全指南 (ISC)](https://www.clinph-journal.com/article/S1388-2457(21)00115-1/)

### 5️⃣ 功能性近红外光谱 (fNIRS)

**主要用途**：
- 运动任务中皮层血氧响应
- 运动学习过程中的皮层可塑性
- 相比 fMRI，允许自由运动

**设备品牌**：
- NIRx
- Hitachi ETG-4000

**注意事项**：
- 头发茂密区域信号质量差 → 选择**前额叶**或**运动皮层**区域
- 运动伪影（Motion Artifact）是主要噪声来源
- 空间分辨率低（相比 fMRI）

---

### 6️⃣ 眼动仪 (Eye Tracker)

**主要用途**：
- 运动规划中的眼球运动（扫视、平滑追踪）
- 视觉注意与运动控制的关系
- 运动学习中的注视模式变化

**设备品牌**：
- EyeLink（高采样率，研究标准）
- Tobii（便携，适合行为实验）

## 🛡️ 设备使用安全规范

### 通用安全原则

1. **培训第一**：未经培训不得擅自操作设备
2. **记录使用日志**：每次使用记录在实验室 Logbook
3. **报告故障**：设备异常立即报告实验室管理员
4. **清洁消毒**：与人体接触的设备每次使用后清洁（尤其 EEG/EMG 电极）

### 数据存储规范

!!! success "数据管理最佳实践"
    ```
    推荐目录结构：
    data/
    ├── raw/           # 原始数据（只读，不修改）
    │   ├── sub-01/
    │   ├── sub-02/
    │   └── ...
    ├── derived/       # 处理后的数据
    │   ├── filtered/
    │   └── features/
    ├── behavioral/    # 行为数据
    └── logs/          # 实验日志
    
    备份策略：
    ├── 本地硬盘（工作副本）
    ├── 实验室服务器（每周同步）
    └── 云端备份（Box/OneDrive，每月同步）
    ```

## 📋 设备预约系统

TAMU KNSM 通常使用在线预约系统（如 Google Calendar 或专用软件）：

- **提前预约**：至少提前 24 小时
- **准时开始和结束**：尊重下一位使用者的时间
- **取消政策**：无法使用时提前取消（至少提前 4 小时）
- **设备培训预约**：联系实验室管理员安排
