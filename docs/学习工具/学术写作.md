# 学术写作

!!! info "为什么学术写作很重要？"
    博士生涯的核心是**发表**。无论你的研究多出色，如果写不出来、发表不出去，就无法被学术界认可。

---

## 📊 学术写作的核心原则

### 1️⃣ IMRaD 结构

绝大多数科学论文遵循 **IMRaD** 结构：

```
Title（标题）
├── Abstract（摘要，通常 150-250 词）
└── Main Text
    ├── Introduction（引言）：为什么要做这个研究？
    ├── Methods（方法）：你做了什么？
    ├── Results（结果）：你发现了什么？
    └── Discussion（讨论）：这些发现意味着什么？
```

### 2️⃣ 写作原则

!!! success "George D. Gopen & John M. Swan: The Science of Scientific Writing"
    **读者预期与语法结构匹配**：
    - 句首是已知信息（读者预期延续上文）
    - 句尾是新信息（读者记忆最强处）
    - 主动语态 > 被动语态（大多数情况）
    - 动词 > 名词化（"we measured" 优于 "measurements were made"）

### 3️⃣ 段落结构

```
Topic Sentence（主题句） → 本段核心观点
├── Supporting Sentence 1（支撑句 1）
├── Supporting Sentence 2（支撑句 2）
├── ...
└── Concluding Sentence（小结句，可选）
```

## ✍️ 写作流程建议

### 阶段 1：论文规划（1-2 周）

```
1. 确定目标期刊（影响因子、读者群、发表周期）
2. 阅读该期刊近 2 年相似文章 5-10 篇
3. 分析这些文章的结构、语调、图表风格
4. 制定 Outline（I 级标题 → II 级标题 → 每个小节 2-3 个要点）
```

### 阶段 2：初稿写作（4-8 周）

!!! warning "Paul Silvia: How to Write a Lot"
    **设定每日写作目标，而不是"有灵感再写"**：
    - 每天固定时间写作（如早上 8:00-9:30）
    - 设定字数目标（如每天 300 词）
    - 先写烂初稿，再修改（"Shitty First Drafts"）
    - 不要边写边改（会打断思路）

**推荐写作顺序**（不一定从 Introduction 开始）：
```
1. Methods（最客观，最容易写）
2. Results（数据已有，描述即可）
3. Figures + Tables（先做好图表，文章骨架就有了）
4. Discussion（基于结果，解释发现）
5. Introduction（最后写，因为你要基于全文来引入）
6. Abstract（全文完成后写）
7. Title（最后起，确保准确反映内容）
```

### 阶段 3：修改与润色（2-4 周）

```
Round 1: 结构与逻辑（大修）
├── 每段是否有清晰的主题句？
├── 段落之间是否有过渡句？
├── 图表是否支撑了文中的论点？
└── 方法部分是否足够详细，可以复现？

Round 2: 句子与用词（中文 → 英文常见错误）
├── 检查中式英语（Chinglish）
├── 主动/被动语态是否合理？
├── 时态是否正确？（方法用过去时，结果用现在时）
└── 检查冠词（a/an/the）、介词（in/on/with）

Round 3: 格式与细节
├── 期刊格式要求（引用格式、图表分辨率）
├── 拼写检查（Grammarly / LanguageTool）
└── 最终校对（打印出来读，或用朗读功能）
```

## 🌐 常见中式英语错误

| 中文思维 | 错误英文 | 正确表达 |
|----------|----------|----------|
| "进行研究" | "conduct research" | "study", "investigate", "examine" |
| "表明" | "the results indicate that..." | "the results suggest that..." |
| "有意义" | "have meaning" | "are meaningful", "have implications for..." |
| "虽然...但是..." | "although... but..." | "although... [no but]" |
| "因为...所以..." | "because... so..." | "because... [no so]" |
| "随着...的发展" | "with the development of..." | 直接说研究对象，不用套话 |

!!! tip "避免套话 (Clichés)"
    - ❌ "With the rapid development of..."（每篇论文都这么写）
    - ✅ 直接说："Recent advances in motor control have..." 
    - ❌ "It is well known that..."
    - ✅ 直接引用："(Smith et al., 2015) showed that..."

## 📋 投稿前检查清单

!!! success "投稿前最后确认"
    - [ ] 所有作者已确认作者顺序和贡献声明
    - [ ] 期刊格式要求已全部满足（引用格式、图表分辨率、页数限制）
    - [ ] Cover Letter 已写好（说明研究意义 + 与期刊匹配度）
    - [ ] 所有图表已在正文中引用（Figure 1, Table 2...）
    - [ ] 补充材料（Supplementary Material）已准备好
    - [ ] 语言已润色（如需要，使用专业润色服务）
    - [ ] 所有作者声明无利益冲突（COI Declaration）
    - [ ] 所有数据/代码已上传开放仓库（如需要）

## 📝 排版实战：LaTeX + Overleaf

### 为什么用 LaTeX？

| | **Word** | **LaTeX (Overleaf)** |
|---|---|---|
| 公式排版 | ❌ 难看、难编辑 | ✅ 专业级美观 |
| 大规模文档 | ❌ 超过 50 页变慢 | ✅ 无压力 |
| 参考文献 | ⚠️ 需插件（Zotero） | ✅ BibTeX 原生支持 |
| 期刊模板 | ⚠️ 需手动调整 | ✅ 官方模板直接套用 |
| 学习曲线 | 低 | 中高 |

**结论**：投稿英文期刊 → 用 Overleaf（LaTeX）；写学位论文/中文文章 → Word 也可。

### Overleaf 快速上手

```
1. 注册：overleaf.com → 用学校邮箱（可能有免费 Premium）
2. 新建项目 → 搜索目标期刊模板（如 "Journal of Neurophysiology"）
3. 直接套用官方模板（格式完全符合期刊要求！）
4. 写作用 .tex 文件，参考文献用 .bib 文件
```

### LaTeX 最小示例（可直接复制）

```latex
\documentclass[12pt]{article}
\usepackage{graphicx}   % 插入图片
\usepackage{cite}         % 引用格式
\usepackage{url}          % 网址格式

\title{My First Motor Control Paper}
\author{Your Name\\\email{<EMAIL_REMOVED>}}
\date{\today}

\begin{document}
\maketitle

\begin{abstract}
This study investigated...
\end{abstract}

\section{Introduction}
Motor learning is... \cite{smith2015}.

\section{Methods}
We recruited 20 participants...

\section{Results}
The main effect was significant ($p < 0.05$)...

\section{Discussion}
Our findings suggest...

\bibliographystyle{apalike}
\bibliography{myrefs}  % 对应 myrefs.bib 文件
\end{document}
```

### 常用 LaTeX 包推荐

| 包名 | 用途 | 代码示例 |
|------|------|----------|
| `graphicx` | 插入图片 | `\includegraphics[width=0.8\textwidth]{fig1.png}` |
| `booktabs` | 漂亮表格 | `\toprule`, `\midrule`, `\bottomrule` |
| `siunitx` | 单位格式 | `\SI{30}{\hertz}`, `\SI{1.5}{\meter}` |
| `amsmath` | 数学公式 | `\frac{a}{b}`, `\sum_{i=1}^n` |
| `xcolor` | 颜色 | `\textcolor{red}{significant}` |

### 参考文献管理（BibTeX）

在 Zotero 中导出 `.bib` 文件，格式如下：

```bibtex
@article{smith2015motor,
  title={Motor learning through the combination of speech and manual gestures},
  author={Smith, Jane and Doe, John},
  journal={Journal of Neurophysiology},
  volume={113},
  number={7},
  pages={2345--2356},
  year={2015}
}
```

正文中引用：`\cite{smith2015motor}` → 自动生成 `[1]` 或 `(Smith & Doe, 2015)`

## 📬 如何应对审稿意见（Review Response）

这是论文发表最关键的环节之一。**审稿人不是敌人，是帮助你提高论文质量的免费顾问。**

### 回复审稿意见的三原则

!!! success "黄金法则"
    1. **逐条回复**，不要遗漏任何一条意见
    2. **态度谦逊**，即使你不同意审稿人的观点
    3. **修改要可见**，在正文修改处用红色标注（或说明在第几行）

### Response Letter 标准格式

```
Dear Editor and Reviewers,

Thank you for your thoughtful comments. We have addressed all concerns below.

=== Reviewer #1 ===

Comment 1:
The authors did not explain why they chose the 100-ms window.

Response:
We appreciate this important point. We chose 100 ms based on...
[修改说明] We have added this justification in the Methods section (Lines 120-125).

--- Changes in manuscript ---
Line 120-125 (updated):
"We used a 100-ms window following Smith et al. (2015)..."

===

=== Reviewer #2 ===
...
```

### 常见审稿意见类型与回复策略

| 审稿意见类型 | 推荐策略 | 示例回复 |
|--------------|----------|----------|
| **要求补充实验** | 尽量做；实在做不到要诚实说明原因 | "We agree this would strengthen the paper. Unfortunately, our lab does not have access to fMRI..." |
| **方法描述不清楚** | 立即补充细节 | "We have expanded the Methods section (Lines 80-95)..." |
| **结论过度解读** | 收敛结论，加限定词 | "We have toned down the conclusion and added 'suggest' instead of 'prove'..." |
| **英语表达问题** | 找专业润色或逐句修改 | "We have carefully revised the language throughout..." |
| **引用遗漏** | 补充引用，不要争辩 | "We thank the reviewer for pointing out this relevant work..." |
