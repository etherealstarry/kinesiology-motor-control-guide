# 文献管理

## 工具简介

文献管理是科研工作中必不可少的一环。好的文献管理工具可以帮助你：

- 📚 **高效收集**：一键保存论文、网页等文献
- 🔍 **快速检索**：全文搜索，快速找到需要的文献
- 📖 **便捷阅读**：支持PDF标注、笔记
- 📝 **自动引用**：插入文献时自动生成引用格式
- 🤝 **团队协作**：与导师、同学共享文献库

## 推荐工具详解

### 1. Zotero（强烈推荐 🌟）

**[Zotero官网](https://www.zotero.org/)**

#### 为什么推荐

1. **完全免费开源**
   - 无功能限制
   - 社区活跃，插件丰富
   - 不用担心某天公司倒闭或涨价

2. **浏览器插件强大**
   - 一键保存网页、PDF、视频等多种资源
   - 自动抓取DOI、ISBN等信息
   - 支持Google Scholar、PubMed、Web of Science等

3. **支持多种文献类型**
   - 不仅支持论文，还支持网页、视频、法律案例等
   - 适合运动控制这种交叉学科

4. **插件生态丰富**
   - ZotFile：PDF管理和标注同步
   - Better BibTeX：增强BibTeX导出
   - Zotero Scihub：一键从Sci-Hub下载PDF
   - 等等...

#### 安装和配置

1. **安装Zotero客户端**
   - 访问 [https://www.zotero.org/download/](https://www.zotero.org/download/)
   - 下载并安装对应系统的客户端

2. **安装浏览器插件**
   - Chrome/Firefox/Edge：安装Zotero Connector
   - 安装后，浏览器工具栏会出现Zotero图标

3. **注册Zotero账号（可选）**
   - 注册后可以同步文献库
   - 免费账户有300MB云空间

4. **安装插件（推荐）**
   - ZotFile：用于管理PDF和标注
   - Better BibTeX：如果用LaTeX写论文
   - Zotero Scihub：一键下载PDF（慎用，注意版权）

#### 基本使用

1. **收集文献**
   - 在Google Scholar搜索论文
   - 点击Zotero Connector图标
   - 选择保存到哪个文件夹

2. **组织文献**
   - 创建集合（Collection）来组织文献
   - 使用标签（Tag）来分类
   - 添加笔记（Note）记录想法

3. **插入引用**
   - Word插件：安装后在Word中会出现Zotero工具栏
   - Google Docs插件：在Google Docs中也能用
   - LaTeX：配合Better BibTeX插件使用

#### 进阶技巧

1. **PDF管理和标注**
   - 安装ZotFile插件
   - 可以在平板电脑上标注PDF，然后同步回Zotero
   - 标注会自动保存到笔记里

2. **利用Sci-Hub一键下载PDF**
   - 安装Zotero Scihub插件
   - 右键文献 → Scihub → Fetch PDF
   - **注意**：请遵守版权法，仅用于合法用途

3. **与Obsidian等笔记软件联动**
   - 使用Zotero Integration插件
   - 可以在Obsidian中直接引用Zotero文献
   - 适合构建知识网络

### 2. Mendeley（备选）

**[Mendeley官网](https://www.mendeley.com/)**

#### 特点

- 免费，有2GB云空间
- PDF自动识别和元数据抓取
- 有社交网络功能，可以看到别人在读什么
- 被Elsevier收购，有些人担心未来

#### 适合人群

- 初学者
- 不需要太多高级功能的人
- 已经在使用Elsevier生态的人

### 3. EndNote（如果有免费授权）

**[EndNote官网](https://endnote.com/)**

#### 特点

- 功能非常强大
- 与Web of Science集成好
- 很多学校和医院提供免费授权
- 界面老旧，学习曲线陡

#### 适合人群

- 有机构免费授权的人
- 医学、生物学领域的研究者
- 不需要经常变换工具的人

## 文献管理最佳实践

### 1. 建立系统的文件夹结构

建议按照以下结构组织：

```
My Library
├── 1. 课程相关
│   ├── 运动控制理论
│   ├── 生物力学
│   └── 研究方法
├── 2. 研究方向
│   ├── 运动学习
│   ├── 神经康复
│   └── 运动表现
├── 3. 论文写作
│   ├── 毕业论文
│   └── 期刊论文
├── 4. 综述和元分析
└── 5. 待读清单
    ├── 高优先级
    └── 低优先级
```

### 2. 善用标签（Tags）

除了文件夹，还要善用标签：

- **方法标签**：#实验研究 #综述 #元分析 #案例研究
- **状态标签**：#待读 #已读 #重要 #参考
- **主题标签**：#运动控制 #运动学习 #生物力学 #神经康复
- **年份标签**：#2020-2024 #2015-2019 #经典

### 3. 养成做笔记的习惯

不要只是保存PDF，要养成做笔记的习惯：

1. **PDF标注**
   - 高亮关键句子
   - 添加文字标注
   - 使用不同颜色区分不同含义

2. **Zotero笔记**
   - 总结论文的主要发现
   - 记录对自己研究的影响
   - 写下批判性思考

3. **与笔记软件联动**
   - 使用Obsidian、Notion等笔记软件
   - 建立文献和笔记之间的双向链接
   - 构建知识网络

### 4. 定期整理文献库

- **每周**：整理本周新增的文献，添加标签和笔记
- **每月**：回顾待读清单，删除不相关的文献
- **每季度**：检查重复文献，合并相同条目

### 5. 备份文献库

- **Zotero**：设置自动同步到云端
- **本地备份**：定期导出整个文献库（.riss格式）
- **PDF备份**：定期备份存储PDF的文件夹

## 常见问题

### Q1：Zotero的云空间只有300MB，不够用怎么办？

**解决方案**：

1. **使用WebDAV同步**
   - 可以使用第三方的WebDAV服务（如Koofr、4shared等）
   - 免费获得更多空间

2. **使用Zotero Storage插件**
   - 将PDF存储到Google Drive、OneDrive等云盘
   - 插件会在Zotero和云盘之间同步PDF

3. **只同步元数据，不同步PDF**
   - 在Zotero设置中，取消"同步附件"
   - PDF保存在本地，需要时再手动下载

### Q2：如何从其他文献管理工具迁移到Zotero？

**迁移步骤**：

1. **从Mendeley迁移**
   - 在Mendeley中导出为RIS或BibTeX格式
   - 在Zotero中导入该文件
   - 注意：PDF需要手动重新关联

2. **从EndNote迁移**
   - 在EndNote中导出为XML格式
   - 在Zotero中导入该文件

3. **通用方法**
   - 导出为RIS格式（最通用）
   - 在Zotero中导入

### Q3：如何批量下载论文PDF？

**注意**：请遵守版权法，仅用于合法用途。

**方法**：

1. **使用Zotero Scihub插件**
   - 安装后，可以一键从Sci-Hub下载PDF
   - 可以批量选择多个文献，右键 → Scihub → Fetch PDF

2. **使用开源论文搜索**
   - [arXiv](https://arxiv.org/) - 物理、数学、计算机科学预印本
   - [bioRxiv](https://www.biorxiv.org/) - 生物学预印本
   - [SocArXiv](https://osf.io/preprints/socarxiv/) - 社会科学预印本
   - [Google Scholar](https://scholar.google.com/) - 搜索免费版本
