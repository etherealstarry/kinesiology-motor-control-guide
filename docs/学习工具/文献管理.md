# 文献管理

!!! info "为什么文献管理很重要？"
    博士生涯你会阅读 **500-1000+ 篇论文**。没有好的文献管理工具，你会陷入"读过但找不到"的困境。

---

## 🤔 Zotero 还是 Obsidian？先搞清楚定位

这两个工具**不是二选一**，而是**各司其职、配合使用**：

| | **Zotero** | **Obsidian** |
|---|---|---|
| **核心定位** | 文献数据库（Library） | 知识笔记本（Notebook） |
| **擅长的事** | 收集文献、管理 PDF、插入引用 | 连接想法、构建知识网络 |
| **自动抓取 DOI/标题/作者** | ✅ 浏览器一键抓取 | ❌ 不支持 |
| **Word/LaTeX 引用插入** | ✅ 原生支持 | ⚠️ 需插件，配置麻烦 |
| **PDF 高亮 → 笔记** | ✅ 原生支持 | ⚠️ 需插件 |
| **双向链接 / 知识图谱** | ❌ 没有 | ✅ 核心功能 |
| **适合场景** | "这篇文献我要在论文里引用" | "这个想法和那篇笔记有关" |

**一句话总结**：
- 用 **Zotero** 管理「文献」——它是你的文献数据库
- 用 **Obsidian** 管理「想法」——它是你的第二大脑
- 两者通过插件打通，文献笔记自动同步

## Zotero：最推荐的选择

### 为什么选 Zotero？

- ✅ **完全免费**（开源）
- ✅ **浏览器插件**：一键抓取网页、PDF、YouTube 视频
- ✅ **Word/PPT 插件**：引用插入无缝衔接
- ✅ **iOS/Android 同步**：手机上也能读文献
- ✅ **插件生态**：Zotero Connector, Better BibTeX, Zotfile...

### 安装与配置

!!! success "安装步骤"
    1. 下载 [Zotero](https://www.zotero.org/download/)
    2. 安装 **Zotero Connector**（浏览器插件）
    3. 注册 Zotero 账号（用于同步）
    4. 安装插件：**Better BibTeX**（生成 citekey）、**Zotfile**（PDF 管理）

### 使用技巧

```
高效文献管理工作流：
├── 文献收集
│   ├── 用 Zotero Connector 一键抓取
│   ├── 手动拖入 PDF（自动识别 metadata）
│   └── 订阅 RSS / Google Scholar 提醒
├── 文献组织
│   ├── 按研究主题建 Collection
│   ├── 用 Tag 标记（#method/behavioral, #status/to-read）
│   └── 用 Color 标记优先级
├── 文献阅读
│   ├── 用 Zotfile 提取 PDF 注释
│   ├── 高亮 → 自动同步到 Note
│   └── 附上自己的理解笔记
└── 文献引用
    ├── Word 插件一键插入引用
        └── 导出 .bib 给 LaTeX / Overleaf 使用
```

### Better BibTeX 配置（LaTeX 用户必看）

在 Zotero 设置中（Better BibTeX 首选项）：

```
Citekey 格式：[auth:lower:regex:^([a-z])`]|[year]`[veryshorttitle:regex:^([A-Z])`]
示例：Shadmehr & Krakauer (2008) → [sk2008motor]
```

## 🔬 进阶：Obsidian + Zotero 知识管理

!!! tip "构建个人知识网络"
    用 **Obsidian** 做文献笔记，通过 **Zotero Integration** 插件连接 Zotero，实现：
    - 每篇论文一个笔记
    - 笔记间双向链接（"这篇用了和那篇相同的方法..."）
    - 知识图谱可视化
    - 永久保存，不受工具迁移影响

### 插件推荐

- **Zotero Integration**（Obsidian 插件）：连接 Zotero
- **Citations**（Obsidian 插件）：在 Obsidian 中搜索和插入引用
- **Templater**：创建统一的文献笔记模板

## 📝 文献阅读技巧

### 三遍阅读法（Three-Pass Approach）

!!! abstract "S. Keshav: How to Read a Paper"
    **第 1 遍（5-10 分钟）**：
    - 读标题、摘要、引言、结论、图表标题
    - 目标：判断这篇论文是否值得继续读
    
    **第 2 遍（1 小时）**：
    - 读全文，跳过证明/细节
    - 标记重要图表、关键结论
    - 目标：掌握论文的核心贡献
    
    **第 3 遍（4-5 小时）**：
    - 复现论文：假设你是作者，会怎么做？
    - 对比作者的假设与你的想法
    - 目标：深度理解，能向别人讲解

### 做笔记的建议

| 方法 | 适合 | 工具 |
|------|------|------|
| **高亮 + 批注** | 快速阅读 | Zotero / PDF Expert |
| **康奈尔笔记法** | 深度理解 | 纸质 / OneNote |
| **文献笔记模板** | 系统性整理 | Obsidian / Notion |
| **卡片笔记 (Zettelkasten)** | 长期知识积累 | Obsidian / Roam Research |

**文献笔记模板**（推荐用 Obsidian 保存）：

```
# [论文标题]

## 基本信息
- 作者：
- 年份：
- 期刊：
- 研究方法：

## 研究问题
（1-2句话概括）

## 方法概要
（关键实验设计）

## 主要发现
- 
- 

## 我的评价
（优点 / 不足 / 对我的研究有什么启发）

## 相关文献
[[另一篇论文笔记]]
```
