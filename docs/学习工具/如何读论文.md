# 如何读论文

!!! info "为什么读论文是博士的核心技能？"
    博士生涯你要读 **500-1000+ 篇论文**。读得懂、读得快、读得有效，直接决定你的研究水平。

---

## 📖 三遍阅读法（Three-Pass Approach）

来自 S. Keshav 的经典方法，被引用 5000+ 次。

### 第 1 遍：快速扫描（5-10 分钟）

**目标：判断这篇论文值不值得精读**

```
读这些部分：
├── Title + Abstract（摘要）→ 整篇论文的浓缩
├── Introduction（引言）最后一段 → 作者自己总结的贡献
├── Section Headers（各级标题）→ 论文结构
├── Conclusion（结论）→ 最终答案
└── References（参考文献）前几篇 → 是否引用了你认识的经典文献
```

**读完第 1 遍，你应该能回答 5 个问题：**
1. 论文的论文类别是什么？（实验、理论、综述、方法改进？）
2. 论文的核心贡献是什么？
3. 实验设计是否合理？
4. 这篇论文对我自己的研究有没有用？
5. 值不值得花 1 小时精读？

> **决定：继续读第 2 遍，还是扔进"待读"文件夹？**

---

### 第 2 遍：理解内容（1 小时）

**目标：掌握论文的所有论点，但不纠结于证明细节**

```
读这些部分：
├── 全文逐字读（除了证明/公式推导）
├── 注意图表！图表是论文的"证据"
│   └── 能否仅凭图表理解论文结论？
├── 标记关键参考文献（待读）
└── 读 Discussion（讨论）→ 作者的自我评价和局限
```

**读完第 2 遍，你应该能：**
- 向别人用 5 分钟讲清楚这篇论文干了什么
- 总结出 3 个优点和 3 个不足
- 判断能不能直接用于自己的研究

> **做笔记！** 用下面的模板。

---

### 第 3 遍：复现论文（4-5 小时）

**目标：深度理解，能向别人讲解，甚至复现结果**

```
这一步要做：
├── 假设你是作者，会怎么做这个研究？
├── 对比你的思路和作者的思路差异
├── 如果作者提供了代码/数据，尝试复现
└── 思考：如果是我做这个研究，哪里可以改进？
```

**读完第 3 遍，你应该能：**
- 在组会上讲这篇论文（被提问也不慌）
- 写出这篇论文的"改进版"研究计划

## 📝 论文阅读笔记模板

推荐用 **Obsidian** 保存每篇论文的笔记：

```markdown
# [论文标题]

## 基本信息
- 作者：
- 年份：
- 期刊/会议：
- DOI：

## 研究问题
（1-2 句话概括：他们想解决什么问题？）

## 方法概要
（关键实验设计，画个简单流程图）

## 主要发现
- 发现 1：
- 发现 2：
- 发现 3：

## 我的评价
- ✅ 优点：
- ❌ 不足：
- 💡 对我的研究的启发：

## 相关文献
- [[另一篇论文笔记]]
- [[方法类似的论文]]
```

## 🎯 不同目的的阅读策略

| 阅读目的 | 推荐遍数 | 时间投入 | 重点 |
|----------|----------|----------|------|
| **找研究方向** | 第 1 遍 | 5-10 分钟/篇 | 快速筛选，广撒网 |
| **写文献综述** | 第 1+2 遍 | 1 小时/篇 | 理解方法和结论 |
| **设计自己的实验** | 第 1+2+3 遍 | 4-5 小时/篇 | 复现，找漏洞 |
| **跟踪领域前沿** | 第 1 遍 | 5 分钟/篇 | 只看 Abstract + Conclusion |

## 🔧 实用工具

### 文献追踪工具

| 工具 | 用途 | 链接 |
|------|------|------|
| **Google Scholar Alerts** | 关键词新论文提醒 | [scholar.google.com](https://scholar.google.com/) |
| **ResearchGate** | 关注作者，获取全文 | [researchgate.net](https://www.researchgate.net/) |
| **Academic Twitter (X)** | 跟踪领域大牛，获取预印本 | 关注领域 Top 5% 学者 |
| **arXiv** | 获取最新预印本（神经科学/机器学习） | [arxiv.org](https://arxiv.org/) |
| **bioRxiv** | 生物/神经科学预印本 | [biorxiv.org](https://www.biorxiv.org/) |

### 论文获取途径

!!! tip "合法获取全文的方法"
    - ✅ 通过 TAMU 图书馆 VPN 访问（[library.tamu.edu](https://library.tamu.edu/)）
    - ✅ 使用 [Unpaywall](https://unpaywall.org/) 浏览器插件（找合法 OA 版本）
    - ✅ 给作者发 Email 要论文（成功率 > 50%）
    - ✅ ResearchGate 上直接 Request full text
    - ❌ 不使用 Sci-Hub（版权风险）

### 如何高效管理阅读列表

```
推荐工作流：
1. 每周固定时间（如周五下午）批量搜索新论文
2. 用 Zotero 一键抓取，打标签 #to-read
3. 用 Notion/Obsidian 建一个 "Reading List" 数据库
4. 按优先级排序：🔴 高优先级 / 🟡 中优先级 / 🟢 低优先级
5. 每读完一篇，移到 "Read" 列，并写笔记
```
