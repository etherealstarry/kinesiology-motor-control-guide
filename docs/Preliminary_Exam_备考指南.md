# Preliminary Examination 备考指南

TAMU Kinesiology — Motor Neuroscience PhD 预备考试（Preliminary Examination）的全网搜索备考经验、方法、资源与策略。

---

## 📋 什么是 Preliminary Exam？

Preliminary Examination（预备考试/资格考）是博士生涯中**最重要的里程碑**——通过后你从"博士生（PhD Student）"晋升为"博士候选人（PhD Candidate / Dissertator）"。

### TAMU 官方定义

> "评估学生是否掌握专业所有领域的主要内容、对文献有充分了解、具备书目研究能力、对研究问题有深入理解并掌握适当的方法论。"

### 通过后的变化

| 之前（PhD Student） | 之后（PhD Candidate / Dissertator）|
|-------------------|----------------------------------|
| 上课为主 | 论文为主 |
| 全职注册 | **每学期仅需 3 学分**研究课程 |
| 上课交学费 | 论文费用（显著更低）|
| 不能毕业 | 可以答辩毕业 |

## 🏛️ TAMU 官方考试规则

> 来源：[TAMU Graduate School — Preliminary Exam](https://grad.tamu.edu/degree-completion/steps-to-a-degree/preliminary-exam.html)

### 资格要求

| 条件 | 说明 |
|------|------|
| **学位计划** | 已获批，且通过考试时剩余评分课程 ≤ 6 学分 |
| **GPA** | 学位计划 GPR ≥ 3.0，累计 GPR ≥ 3.0 |
| **注册状态** | 考试学期必须在册 |
| **DPSS** | 无未处理请求 |

### 时间要求

| 节点 | 说明 |
|------|------|
| **申请提交** | 考试日期前 **至少 10 个工作日** 在 ARCS 系统提交 |
| **完成报告** | 考试完成后 **10 个工作日内** 由系里向研究生院报告 |
| **有效期限** | 通过后 **4 年内** 必须完成所有学位要求，否则成绩作废（⚠️ 别拖！） |

### 委员会规则

| 规则 | 说明 |
|------|------|
| **替代委员** | 最多 1 人，**导师不可替代** |
| **投票通过** | 委员会**全体正面评价**，最多允许 1 票反对 |
| **系外替代** | 如原委员来自系外，替代者也应来自系外 |

### 失败与重考

> 具体政策参考 TAMU Graduate Catalog。通常允许重考一次。有些系允许在特定条件下重考第二次。

## 📝 典型的考试格式

> TAMU KNSM Motor Neuroscience 的具体格式以系内 Graduate Handbook 为准，以下是基于 TAMU 工程院 + 同领域其他学校 Kinesiology PhD 的综合参考：

### 总体结构：Written + Oral

```
笔试（Written Exam）
├── 时间：通常 2-5 天内完成
├── 题目：4 道多部分综合题（每道 4 小时）
├── 出题：导师 2 题 + 每位委员会成员各 1 题
└── 形式：开卷/闭卷由导师决定

口试（Oral Exam）
├── 时间：笔试后 2 周内
├── 时长：2-3 小时
├── 内容：基于笔试答案 + 研究 proposal
└── 形式：委员会全体出席
```

### 笔试可能的出题方向

| 出题人 | 可能方向 | 覆盖课程 |
|--------|----------|----------|
| **导师** | 你的研究方向核心理论 + 方法论 | 与研究高度相关 |
| **委员会成员 1** | 运动控制/运动学习理论 | KINE 606, 640 |
| **委员会成员 2** | 发展/自组织/动态系统 | KINE 641, 642 |
| **委员会成员 3** | 研究方法/统计/实验设计 | KINE 601, 研究统计 |

### ⚠️ 重要禁令

> **严禁使用生成式 AI（ChatGPT 等）**，除非导师明确书面许可。这是学术诚信的底线。

## 🎯 备考策略

### 阶段一：提前 6-12 个月 — 建立基础

```
目标：确保你对领域核心内容有全局掌握

行动：
1. 确认你的 committee 成员名单（至少提前 6 个月）
2. 和每位 committee member 单独开会，问清楚：
   ├── 他们最看重什么能力？
   ├── 他们可能会出什么方向的题？
   ├── 他们推荐哪些必读论文/教材？
   └── 他们对 research proposal 的期望是什么？
3. 整理各门核心课程的笔记（KINE 606/640/641/642）
4. 开始汇编 "Prelim Reading List"（见下方模板）
```

### 阶段二：提前 3-6 个月 — 深度准备

```
目标：建立领域知识体系 + 撰写 research proposal 初稿

行动：
1. 按 committee member 分别创建阅读清单
2. 每周精读 2-3 篇核心论文 + 做总结笔记
3. 开始撰写 research proposal（至少 3 个月时间）
   ├── Background & Significance
   ├── Specific Aims
   ├── Methods
   ├── Expected Results & Alternative Hypotheses
   └── Timeline
4. 组织或加入 Prelim Study Group（2-4 人）
5. 和高年级已通过 Prelim 的学生聊经验
```

### 阶段三：考前 1-2 个月 — 模拟冲刺

```
目标：模拟训练 + 查漏补缺

行动：
1. 做 mock written exam（让导师或 lab mate 出题）
2. 做 mock oral defense（至少 2 次，录视频复盘）
3. 给 lab mates/friends 做 research proposal presentation
4. 整理 "最可能被问到的问题清单" 并准备答案
5. 检查所有后勤细节（考场、电脑、笔记权限等）
```

## 📝 Research Proposal 撰写指南

### Proposal 是你 Prelim 的核心

笔试 + 口试的很多问题会围绕你的 proposal 展开。committee 通过 proposal 考察：
- 你是否能提出有价值的科学问题？
- 你是否有可行的研究计划？
- 你是否考虑了备选方案和局限性？

### Proposal 结构（按 NIH R21 格式）

```
1. Specific Aims（1 页）
   ├── 核心假设
   ├── Aim 1: ...
   ├── Aim 2: ...
   └── Aim 3: ...

2. Background & Significance（2-3 页）
   ├── 文献综述（展示你对领域的掌握）
   ├── 研究空白（gap）
   └── 为什么你的研究重要？

3. Preliminary Data（如有，1-2 页）
   └── 你 rotation 或已有实验的初步结果

4. Research Approach（4-6 页）
   ├── Aim 1 Methods
   │   ├── 实验设计
   │   ├── 被试与样本量
   │   ├── 数据分析计划
   │   ├── 预期结果
   │   └── 替代假设与备选方案
   ├── Aim 2 Methods（同上结构）
   └── Aim 3 Methods（同上结构）

5. Timeline（1 页）
   └── 每学期/每季度的研究计划

6. References
   └── 不必穷尽，但关键论文必须引用
```

### ⚠️ 最常见的 Proposal 踩坑

| # | 坑 | 避免方法 |
|---|-----|----------|
| 1 | **假设不可证伪** | 确保每个假设都能被实验数据支持或推翻 |
| 2 | **没有备选方案** | 每个 aim 都写 "如果结果不是预期那样，下一步做什么" |
| 3 | **样本量太小** | 提前做 power analysis，展示你考虑了统计效力 |
| 4 | **忽略局限性** | 主动承认方法的局限，展示批判性思维 |
| 5 | **太多 aim** | 2-3 个 specific aims 是最优数量 |
| 6 | **文献引用老旧** | 确保引用了近 3 年的相关研究（展示你跟进前沿） |

## 🗣️ 口试准备（Oral Defense）

### 口试流程（典型 2-3 小时）

```
1. Opening：你的 research proposal 陈述（20-30 分钟）
2. Committee Q&A（1.5-2 小时）
   ├── 关于 proposal 的问题
   ├── 关于笔试答案的问题
   ├── 基础理论问题
   └── "如果...你会怎么做？" 型问题
3. Deliberation：委员会闭门讨论（15-30 分钟）
4. Result：告知结果
```

### 口试高频问题清单

#### 关于你的研究

- "你的核心假设是什么？为什么这个假设值得验证？"
- "如果 Aim 1 失败了，你的备选方案是什么？"
- "你的实验设计中最薄弱的环节是什么？"
- "你如何定义和测量你的因变量？有什么替代测量方式？"
- "你的样本量是多少？为什么选择这个数量？power analysis 是怎样的？"

#### 关于基础理论

- "OFC 和 Active Inference 的核心区别是什么？"
- "解释内部模型（internal model）的概念，并举例说明。"
- "运动学习的适应性（adaptation）和技能学习（skill learning）有什么区别？"
- "动态系统理论如何解释运动发育？"

#### "如果...怎么办" 型问题

- "如果你发现结果和预期完全相反，你怎么解释？"
- "如果你的方法存在混淆变量，你会怎么控制？"
- "如果有人重复你的实验但得不到相同结果，可能是什么原因？"

### 口试策略

!!! tip "口试黄金法则"

    1. **不知道就说不知道**：猜错比说"我不知道"糟糕 100 倍
    2. **思考可以出声**："让我想一下...这个问题可以从 X 和 Y 两个角度来考虑..."
    3. **把问题变对话**：不要像背诵答案，而是和 committee 讨论
    4. **控制节奏**：如果一个问题回答太久，简短地说 "这是我能想到的主要内容，还有更多但可能不需要全部展开"
    5. **准备一张"救命卡"**：列出你最熟悉的 5 篇论文，碰见答不上来的问题时可以说 "虽然我不能直接回答这个问题，但相关的文献（引用某论文）可能有线索..."
    6. **committee 打断你不一定是坏事**：有时他们只是对某个点特别感兴趣

## 👥 Committee 关系管理

### 选择 Committee Member 的策略

| 角色 | 建议选择 | 原因 |
|------|----------|------|
| **Chair（导师）** | 你的 primary advisor | 必须的 |
| **Member 1** | 系内 Motor Neuroscience 教授 | 领域专家，出你研究方向的核心题 |
| **Member 2** | 系内或系外方法论专家 | 负责质疑你的实验设计和统计 |
| **Member 3** | 系外教授（可选但推荐） | 提供外部视角，有时更容易沟通 |

### 和 Committee 的沟通策略

!!! note "Prelim 前的沟通清单"
    - [ ] 至少提前 6 个月确认 committee 名单
    - [ ] 和每位成员单独开一次 "pre-prelim meeting"，问清楚期望
    - [ ] 在正式考试前 2-3 周提交 proposal 给全 committee（让他们有时间读）
    - [ ] 如果某个 member 特别难搞，多和 TA 单独沟通，理解 TA 的关切点

## 🧘 心理与健康管理

Prelim 是博士生涯中压力最大的节点之一。以下不是可选建议，是生存必须：

| 策略 | 具体行动 |
|------|----------|
| **睡眠** | 考前一周每晚 7-8 小时，比多读两篇论文重要得多 |
| **运动** | 每天至少 30 分钟（跑步/散步），运动控制专业的你不会不知道运动对大脑的好处 |
| **社交支持** | 找已经通过 Prelim 的前辈聊，他们的"我也经历过"比任何鸡汤都有用 |
| **休息日** | 每周至少休 1 天完全不学习的日子 |
| **心理咨询** | TAMU Counseling Services 免费且保密，不要有心理负担 |
