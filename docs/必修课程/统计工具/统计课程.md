## STAT 652: Statistics for Experimenters I

!!! success "最推荐的第一门统计课"
    **这是实验科学研究生的最佳入门统计课程**，内容覆盖实验设计、方差分析、多重比较等实验研究者最需要的内容。

### 课程基本信息

| 项目 | 内容 |
|------|------|
| 课程号 | STAT 652 |
| 学分 | 3 credits |
| 先修课 | 无（需本科基础统计知识） |
| 典型开课 | Fall/Spring |
| 考试 | 通常有期中考试 + 期末项目 |

### 主要学习内容

1. **实验设计基础**
   - 完全随机设计 (CRD)
   - 随机区组设计 (RBD)
   - 因子设计 (Factorial Design)
   - 嵌套设计 (Nested Design)

2. **方差分析 (ANOVA)**
   - 单因素 ANOVA
   - 双因素 ANOVA
   - 重复测量 ANOVA（**运动控制研究核心！**）
   - 混合效应模型入门

3. **多重比较校正**
   - Tukey HSD
   - Bonferroni 校正
   - False Discovery Rate (FDR)

4. **功效分析 (Power Analysis)**
   - 样本量估算
   - 统计功效计算
   - G*Power 软件使用

5. **R 语言实践**
   - 数据清洗与可视化
   - `afex` / `emmeans` 包进行 ANOVA
   - 可重复性报告 (RMarkdown / Quarto)

### 为什么运动神经科学方向特别需要这门课？

> 运动控制研究大量使用**重复测量设计**（同一受试者在多个时间点/条件下的测量），STAT 652 的重复测量 ANOVA 和混合效应模型内容直接适用于你的数据分析需求。

### 推荐教材

- 📖 **Primary Text**: Montgomery, D. C. *Design and Analysis of Experiments*. Wiley.
- 📖 **R 实践**: Wickham, H. & Grolemund, G. *R for Data Science*. O'Reilly. [免费在线](https://r4ds.hadley.nz/)
- 📖 **进阶参考**: Kutner, M. H. et al. *Applied Linear Statistical Models*. McGraw-Hill.

### 学习建议

```
时间分配建议：
├── 课前（15min）：预习章节，标记不懂的概念
├── 上课（75min）：专注听讲，跟上 R 演示
├── 课后（2-3h）：完成每周作业，用 R 复现课堂例子
└── 期末前（1周）：整理 Cheat Sheet，复习所有 ANOVA 类型
```

!!! tip "实用技巧"
    尽早开始用 R 做作业，不要依赖 SPSS 点击操作。R 的代码思维会让你更深刻理解统计模型。

## STAT 608: Regression Analysis

### 课程概述

线性回归模型的系统学习，适合已经修过基础统计、想深入回归方法的同学。

### 主要学习内容

- 简单线性回归
- 多元线性回归
- 模型诊断（残差分析、异方差性）
- 变量选择方法（逐步回归、LASSO）
- 逻辑回归（二分类/多分类）
- 广义线性模型 (GLM)

### 运动神经科学应用

- 运动表现预测模型（基于生理指标）
- 脑电信号与行为指标的回归分析
- 步态参数的预测模型

## STAT 638: Introduction to Statistical Learning

### 课程概述

机器学习方法的统计基础，适合有编程基础、对数据科学感兴趣的同学。

### 主要学习内容

- 监督学习 vs 非监督学习
- 正则化方法（Ridge / LASSO / Elastic Net）
- 树模型与随机森林
- 支持向量机 (SVM)
- 交叉验证与模型评估
- 神经网络入门

### 推荐教材

- 📖 **强烈推荐**: James, G. et al. *An Introduction to Statistical Learning*. Springer. [免费在线](https://www.statlearning.com/)
  - 有 R 和 Python 两个版本！
  - 非常适合自学

## STAT 636: Statistical Computing

### 课程概述

统计计算的编程实现，需要较好的编程基础。主要使用 R 和 C++（通过 Rcpp）。

### 主要学习内容

- R 语言高级编程
- 数值优化算法
- MCMC 方法（Markov Chain Monte Carlo）
- Bootstrap 与 Jackknife
- Rcpp 加速计算

---

## STAT 654: Bayesian Statistics

### 课程概述

贝叶斯统计理论，适合理论取向较强的研究者。

### 主要学习内容

- 贝叶斯定理与先验/后验分布
- MCMC 采样（Gibbs Sampling, Metropolis-Hastings）
- 层次模型 (Hierarchical Models)
- 贝叶斯因子与模型比较
- Stan / JAGS 软件使用

### 运动神经科学应用

- 个体差异建模（层次贝叶斯模型）
- 脑信号分析的贝叶斯方法
- 运动学习过程的贝叶斯建模

!!! warning "重要提醒"
    TAMU Kinesiology 博士生需要**通过统计资格考试**，通常在修完第一门统计课后参加。考试形式通常为开卷，重点考察实验设计和 ANOVA 的实际应用能力。
