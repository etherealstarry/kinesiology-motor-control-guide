# 实验设计与统计

## 课程简介

**实验设计与统计**是运动机能学研究的核心方法课程，教授如何设计严谨的实验、收集有效数据、使用统计方法分析数据、并正确解释结果。本课程是进行科学研究和阅读学术论文的基础。

**先修知识**：基础统计学、研究方法基础

**参考课时**：3-4学分，通常为期一学期

## 学习建议

### 学习方法

1. **理解统计概念而非死记公式**
   - 统计是工具，重点是**理解何时使用何种统计方法**
   - 理解p值、置信区间、效应量的实际意义
   - 避免"p-hacking"和"数据窥探"

2. **结合软件实操**
   - 理论学习后，立即用统计软件（SPSS、R、Python）实操
   - 使用运动科学领域的真实数据集练习
   - 学会解读软件输出结果，而不仅仅是得到p值

3. **从研究论文中学习**
   - 阅读顶级期刊（如Medicine & Science in Sports & Exercise）的方法与结果部分
   - 分析作者如何设计实验、选择统计方法、呈现结果
   - 思考：如果是你，你会如何设计这个研究？

4. **动手设计实验**
   - 课程项目：设计一个完整的研究，从提出问题到数据分析
   - 考虑实际问题：样本量、随机化、盲法、控制变量
   - 寻求教授或研究生的反馈

### 重点主题学习指南

#### 1. 研究设计
- **核心概念**：实验设计、准实验设计、相关性设计、纵向设计
- **学习要点**：
  - 不同设计类型的优缺点和适用场景
  - 内部效度 vs 外部效度
  - 如何控制混淆变量
- **实践应用**：为自己的研究问题选择合适的研究设计。

#### 2. 描述统计
- **核心概念**：均值、中位数、标准差、方差、正态分布
- **学习要点**：
  - 如何选择合适的集中趋势和离散趋势指标
  - 如何呈现数据（表格、图形）
  - 偏态分布的处理
- **实践应用**：为自己收集的数据计算描述统计量并制作图表。

#### 3. 推断统计
- **核心概念**：假设检验、p值、I型错误、II型错误、统计功效
- **学习要点**：
  - 零假设和备择假设的设置
  - 如何解释p值（不是"效应大小"！）
  - 统计功效和计算所需样本量
- **实践应用**：为自己设计的研究计算所需样本量。

#### 4. ANOVA和回归分析
- **核心概念**：单因素ANOVA、重复测量ANOVA、多元回归、逻辑回归
- **学习要点**：
  - 何时使用ANOVA vs t检验 vs 回归
  - 如何检查假设条件（正态性、方差齐性、独立性）
  - 事后检验的选择和解读
- **实践应用**：使用真实数据集练习ANOVA和回归分析。

## 课程项目示例

### 项目1：完整研究设计
- **目标**：设计一个完整的研究，从提出问题到数据分析计划
- **方法**：
  1. 选择一个研究问题（如"不同热身方式对短跑起跑时间的影响"）
  2. 进行文献回顾，提出假设
  3. 设计实验：被试、分组、干预、测量、控制
  4. 选择统计方法，计算所需样本量
  5. 撰写研究计划书
- **学习收获**：掌握研究设计的完整流程。

### 项目2：数据分析实战
- **目标**：使用统计软件分析真实数据集
- **方法**：
  1. 从开放数据集（如[OpenML](https://www.openml.org/)）下载运动科学相关数据
  2. 进行探索性数据分析（EDA）
  3. 根据研究问题选择合适的统计方法
  4. 进行分析，并正确解读结果
  5. 撰写分析报告
- **学习收获**：掌握统计软件的实际应用。

## 常见问题解答

### Q1：统计学很难，我数学不好能学好吗？
**A**：现代统计学习更强调**概念理解**和**软件应用**，而非复杂数学推导。重点掌握：
1. **何时使用何种统计方法**（决策树）
2. **如何解读结果**（p值、置信区间、效应量）
3. **如何避免常见错误**（p-hacking、多重比较）
- 数学基础薄弱可以从"Practical Statistics"类教材入手。

### Q2：SPSS、R、Python该学哪个？
**A**：取决于你的需求和职业规划：
1. **SPSS**：用户友好，适合初学者和临床应用。但功能有限，且付费。
2. **R**：免费、强大、学术圈主流。学习曲线陡峭，但资源丰富。
3. **Python**：免费、灵活、工业界主流。适合大数据和机器学习。
- **建议**：先学SPSS打基础，再学R或Python进阶。

### Q3：如何避免过度解读统计结果？
**A**：
1. **区分统计显著性和实际显著性**：p < 0.05 不代表效应"很大"或"很重要"
2. **报告效应量**：如Cohen's d、η²、R²
3. **报告置信区间**：比p值提供更多关键信息
4. **考虑多重比较**：使用Bonferroni校正等方法
5. **复现研究**：尝试复现关键结果

## 总结

实验设计与统计是运动机能学研究的**核心技能**。通过系统学习，你将掌握：

- 如何设计严谨的实验
- 如何选择合适的统计方法
- 如何正确解读和呈现结果
- 如何批判性评估他人的研究

**下一步**：在掌握实验设计与统计后，可以继续学习**高级统计方法**（如多元统计、结构方程模型）、**元分析**等进阶课程，或者将所学应用到**实际研究项目**中。
