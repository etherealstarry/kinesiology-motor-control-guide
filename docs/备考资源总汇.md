## 📝 一、学习经验与高分攻略（Aggie 社区精华）

### 来自 r/aggies 的高分通用法则

!!! quote "TAMU 学长学姐的跨学科高分智慧"
    - **主动回忆 > 被动复习**：多位 Aggie 强调 "re-reading creates false confidence"——在白板上默写概念图、合上笔记本自己出题考自己
    - **Office Hours 是最被低估的提分工具**：教授/TA 在 OH 里透露的"这个不会考"或"这个大概率考"就是最精准的考点地图
    - **PPT >> 教材**：TAMU Kinesiology 学生反馈大多数考试 80-90% 内容直接来自授课 PPT，教材用于补充理解不懂的概念
    - **5 天分散复习法**：TAMU ASC 官方推荐——把材料分成 4 块，每天 2-2.5 小时复习，每天覆盖前一天的旧内容 + 新内容
    - **学习小组 ≠ 社交**：找 2-3 个同班认真的同学组队，每人负责整理一章资料，资源共享
    - **Syllabus 是第一手情报**：开学第一天锁定考试日期、评分权重的真实分布、late policy

ASC 是 TAMU 学业提升的核心部门，所有服务对注册学生**完全免费**。

## ✅ 四、备考建议清单（Actionable To-Do List）

### 🟢 现在就能做的事（开学前）

| # | 任务 | 预计耗时 | 为什么重要 |
|---|------|:---:|------|
| 1 | 📥 下载所有教材的免费 PDF 版本（查询 Archive.org / Library Genesis） | 1h | 开学前预读前 3 章能减少 50% 的学期初压力 |
| 2 | 🔐 注册 Quizlet 账号，搜索 "TAMU [课程号]" 保存所有现成闪卡 | 30min | 现成资源，直接用，不自己做 |
| 3 | 📱 下载 Navigate App 并登录，熟悉 Advisor 预约流程 | 10min | Q-Drop / 选课 / 学位规划的重要工具 |
| 4 | 📂 在 Google Drive 建每门课的文件夹结构 | 15min | 组织化 = 效率提升 |
| 5 | 📝 打开 Canvas / Howdy，确认每门课的 Syllabus | 30min | 锁定考试日期、评分权重 |
| 6 | 📆 把所有考试日期导入日历（Google/Apple Calendar） | 10min | 可视化整个学期的"考试密度" |

### 🟡 开学后的复习节奏

```
每周例行：
  ├── 周日晚上：预览下周每门课的 PPT（各 15 min）
  ├── 课后当天：花 30 min 把新概念做成闪卡
  ├── 周五下午：本周学的内容全部 Quizlet 过一遍
  └── 周末：做 1 套 problem set / 复习题

考试前 2 周：
  ├── Week -2：整理 A4 公式表 / 概念对比表
  ├── Week -1：刷往届题目 + 标记弱项 → 重点攻
  └── 考前 72h：按 ASC 5 天计划走（见下方）
```

### 🔴 考前 72 小时极速清单

1. ✅ 打开 ASC 5 天学习计划模板，填入考试范围
2. ✅ Quizlet 闪卡切换为 "Write Mode"（手打答案，不要只是翻卡）
3. ✅ 一张白纸，不看书默写所有核心概念 + 公式
4. ✅ 开卷考试：在教材上贴好彩色便签标注关键页码
5. ✅ 整理 3 个"一定会考但你还没彻底弄懂"的概念 → 去 OH
6. ✅ 睡眠 > 通宵——运动科学和统计学都需要清醒的大脑
