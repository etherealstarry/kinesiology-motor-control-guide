## 🎓 KINE 601：Proseminar in Kinesiology

### 课程性质

博士新生必修的**系内学术研讨课**（1学分），由 **John J. Buchanan**（Program Chair）整体协调。

**核心目标**：帮助你从"学生"过渡到"学者"——了解系内教授研究、学会做学术报告、建立学术社交网络。

### 📋 每周内容与应对策略

| 阶段 | 内容 | 你的行动 |
|------|------|----------|
| **Week 1-2** | 项目介绍、博士时间线、系内规章 | 搞清楚 Prelim 时间线、学位要求、rotation 规则 |
| **Week 3-8** | 教授轮流分享研究方向 | 提前查背景→准备1-2个有深度的问题→提问留下印象 |
| **Week 9-10** | RCR（负责任研究行为）+ IRB 伦理培训 | 完成 CITI Program 在线模块（课外） |
| **Week 11-13** | **学生报告**（每人15-20分钟） | 提前2周准备 slides → lab mate 试讲2次 → 严格计时 |
| **Week 14-15** | Grant writing workshop | 带上你的 proposal 草稿或研究想法 |

### 🎯 关键策略

#### 如何让教授记住你（每周seminar的提问策略）

1. **课前15分钟**：Google 演讲教授的名字 + 最新论文（近2年），读摘要即可
2. **问什么才有深度？** 不是"您招学生吗"，而是：
   - "您在 paper X 里用 force-field adaptation，为什么选这个而不是 visuomotor rotation？"
   - "您的研究对 clinical population 有什么 implications？"
3. **记笔记不只记内容**：还记教授的 presentation style、slide 设计、Q&A 方式——这些都是你以后做报告的免费模板

#### 你的学生报告

!!! danger "这是你在全系面前的第一个专业印象"

    **内容选择优先级**：
    - 最佳：rotation lab 的 preliminary data
    - 次佳：研究兴趣 + 初步 research question + 为什么有趣
    - 避免：硕士/本科做的完全不相关的东西
    
    **时间线**：
    - 提前2周：确定主题 + 大纲
    - 提前1周：做完 slides → lab mates 试讲
    - 提前3天：根据反馈修改 → 再试讲（录视频自己看）
    - 当天：提前15分钟到，测试投影
    
    **常见翻车**：
    - ❌ Slides 字太多 → 每页最多3-4个要点
    - ❌ 时间失控 → 练3遍，严格计时
    - ❌ 被问到不会的问题只说"I don't know" → 应该说"I haven't considered that angle, but based on what we know..."

### 📝 详细学习笔记（Week-by-Week）

#### Weeks 1-3：神经解剖基础

**目标**：能不看图画出运动控制系统的完整连接图

```
推荐策略：
├── 用 Anki 做"功能→区域"闪卡
│   ├── 正面："哪个脑区直接投射到脊髓控制手指运动？"
│   └── 背面："M1（初级运动皮层），通过皮质脊髓束"
├── 用 NeuroSynth / BrainMap 可视化运动相关脑区
└── 画一张从皮层到脊髓的完整连接图（手绘即可）
```

**自测问题**：
- 切除 PMd 和切除 PMv，动作障碍有什么不同？
- SMA 和 Pre-SMA 在序列学习中的分工是什么？
- 小脑浦肯野细胞的"简单 spike"和"复杂 spike"分别编码什么？

**关键阅读**：Kandel 第33-38章（运动系统） + Lemon (2008) "Descending pathways" *Ann Rev Neurosci*

#### Weeks 4-6：感觉运动整合

**目标**：理解贝叶斯模型，能解释感觉重校准实验

**核心直觉**：当戴上左右反转棱镜眼镜——视觉告诉你手在A位置，本体感觉告诉你手在B位置→两个感觉冲突→大脑做了一个"加权平均"来决定手到底在哪→经过20分钟适应，感觉权重被重新分配。

```
推荐策略：
├── 先理解概念，不要被贝叶斯公式吓到
│   └── 核心直觉：σ²越小（越可靠）→ 权重越大
├── 用 Python 写一个 simple sensory integration simulation
└── 重点理解 van Beers 实验 → 感觉重校准的时间进程
```

**自测问题**：
- 戴上棱镜眼镜20分钟后摘掉，手会往哪边偏？为什么？
- 老年人感觉重校准为什么比年轻人慢？

**关键阅读**：Wolpert et al. (2011) *Nat Rev Neurosci* Figure 1（感觉运动整合完整框架）

#### Weeks 7-9：内部模型与最优控制

**目标**：区分正向模型 vs 逆向模型，理解 OFC

```
关键区分：
├── 正向模型（Forward Model）：motor command → predicted sensory consequence
│   类比："如果我按这个键，屏幕会显示什么？"
├── 逆向模型（Inverse Model）：desired state → required motor command
│   类比："我想屏幕上显示X，应该按哪个键？"
└── OFC：正向模型 + 逆向模型 + 反馈增益 → "只纠正重要的错误"
```

**核心实验**：Flanagan & Wing (1997) → 拿起不同重量物体时握力在接触前就已调整（前馈控制/正向模型的直接证据）

**关键阅读**：Todorov & Jordan (2002) Figure 1 + Shadmehr & Krakauer (2008) "Error-based learning" 章节

#### Weeks 10-12：运动学习机制

| 维度 | Adaptation | Skill Learning |
|------|-----------|----------------|
| 时间尺度 | 分钟~小时 | 天~周~月 |
| 机制 | 参数调整 | 新模型形成 |
| 学习信号 | 感觉预测误差 | 任务成功/奖励 |
| 主要脑区 | 小脑 | 皮层-基底节-小脑网络 |
| 经典范式 | Force-field adaptation | Sequence learning |
| 去适应后 | After-effect | 技能保持 |

**关键实验**：Walker et al. (2002) → 睡眠组记忆保持显著优于清醒组（睡眠依赖巩固的证据）

#### Weeks 13-16：运动障碍

| 障碍 | 病理 | 计算模型解释 |
|------|------|-------------|
| 帕金森病 | 黑质多巴胺能神经元退化 | 强化学习信号受损→动作选择困难 |
| 中风 | 局部神经元死亡 | 神经可塑性→未受损区域可"接管"部分功能 |
| 小脑损伤 | 小脑退化/损伤 | 正向模型受损→无法预测运动后果 |

**5篇框架论文（读懂就有全局观）**：

| # | 论文 | 精读重点 |

| 1 | Wolpert et al. (2011) *Nat Rev Neurosci* | Figure 1（框架总图） |

## 🔬 KINE 640：Motor Neuroscience II

> 已有详细课程笔记：[Motor-Neuroscience-II.md](必修课程/核心课程/Motor-Neuroscience-II.md)

### 课程性质

KINE 606 的进阶。核心转变：从"理解已有理论"到**"批判理论并设计方案测试"**。

### 👨‍🏫 可能授课教授

最可能是 **Yuming Lei**（TMS/tDCS 专长 + 神经可塑性）或 **Yue Du**（心理物理学 + 计算建模）。

### 📝 详细学习笔记

#### 三大理论辩论（考点集中区）

**辩论1：FBS vs OFC**

| 维度 | FBS (Feldman) | OFC (Todorov) |
|------|---------------|---------------|
| 运动轨迹 | 涌现属性 | 最优计算 |
| 扰动处理 | λ model | 反馈增益 K_t |
| 目前共识 | 历史价值 | 主流范式 |

**辩论2：OFC vs Active Inference**

| 维度 | OFC | Active Inference |
|------|-----|-----------------|
| 最小化什么 | cost function | 自由能 / surprise |
| 理论基础 | 最优控制 | 贝叶斯推理 |
| 不确定性 | 加权在 cost 中 | 主动采样减少（epistemic action） |

**辩论3：各脑区在 OFC 中的角色**

| 脑区 | OFC 预测的功能 | 关键证据 |
|------|---------------|----------|
| M1 | 实施最优反馈增益 | TMS 改变反馈增益 |
| 小脑 | 正向模型（预测感觉后果） | 小脑损伤→error correction 缺失 |
| 基底节 | 动作选择 + vigor | PD 患者 optimality 受损 |
| 顶叶 | 多感觉状态估计 | 顶叶损伤→感觉权重异常 |

#### 神经调控技术学习路径

```
TMS（经颅磁刺激）
├── 原理：法拉第电磁感应 → 皮层去极化
├── 指标：MEP, RMT, SICI/ICF
├── 应用：因果推断(virtual lesion)、皮层映射、rTMS(LTP/LTD类效应)
└── 必读：Rossi et al. (2009) 安全指南

tDCS/tACS（经颅电刺激）
├── 原理：弱电流调节皮层兴奋性
├── 优势：可做 sham control（双盲），廉价
├── 争议：效应量小，重复性存疑
└── 工具：SimNIBS → 电场仿真

EEG
├── 指标：ERP, ERD/ERS, 时频分析, 相干性
├── 必读：Cohen (2014) "Analyzing Neural Time Series Data"
└── 工具：MNE-Python, FieldTrip

fMRI
├── 方法：GLM, MVPA, RSA, DCM
├── 局限：BOLD ≠ 神经元活动
└── 工具：SPM, FSL, Nilearn
```

## 👶 KINE 641：Developmental Motor Neuroscience

> 已有详细课程笔记：[Motor-Neuroscience-Developmental.md](必修课程/核心课程/Motor-Neuroscience-Developmental.md)

### 课程性质

从**全生命周期**视角研究运动控制——胎儿期→婴儿期→儿童期→成年期→老年期。这不只是"运动控制+年龄变量"，而是用发展的眼光重新审视理论本身。

### 👨‍🏫 可能授课教授

**Carl Gabbard**（Professor Emeritus）或轮换教授。Gabbard 是终身运动发展研究的权威。

### 📝 核心学习笔记

#### 关键理论：Thelen 的婴儿踏步实验

这是用"发展就是一个自然实验"来检验理论的经典案例：

| 现象 | 传统解释（错） | DST解释（对） |
|------|---------------|-------------|
| 新生儿有踏步反射 | 先天运动程序 | 脊髓 CPG 出生时已存在 |
| 2-3个月"消失" | 神经回路退化 | **腿变重了**（脂肪增加），力量不足以对抗重力 |
| 放水里"恢复" | — | 浮力减小重力约束 → CPG 重新表现 |

**核心启示**：运动行为变化不一定是因为大脑"学会了"或"忘记了"——往往只是身体+环境+任务的约束条件变了。

#### 全生命周期关键知识点

| 阶段 | 核心问题 | 关键概念 |
|------|----------|----------|
| 胎儿期 | 自发性运动何时出现？ | General Movements (GMs) 是神经系统"自检" |
| 婴儿期 | 反射→自主运动的转变 | 不是"反射消失"，是皮层抑制的成熟 |
| 儿童期 | 基本动作技能(FMS)发展 | DCD（发展性协调障碍，患病率5-6%） |
| 成年期 | 运动表现的巅峰与维持 | 刻意练习 vs 10,000小时规则 |
| 老年期 | 为什么变慢变不稳？ | 肌少症 + 感觉重加权 + 共同收缩增加(保护性但低效) |

#### 临床评估工具

| 工具 | 年龄 | 用途 |
|------|------|------|
| **GMA** (General Movement Assessment) | 0-5月 | 脑瘫早期筛查（敏感性>90%） |
| **AIMS** (Alberta Infant Motor Scale) | 0-18月 | 婴儿运动发育 |
| **TGMD-3** (Test of Gross Motor Development) | 3-10岁 | 大肌肉动作 |
| **MABC-2** (Movement ABC-2) | 3-16岁 | 运动能力标准化测试 |

## 🔄 KINE 642：Self-Organization in Movement

> 已有详细课程笔记：[Self-Organization.md](必修课程/核心课程/Self-Organization.md)

### 课程性质

这门课挑战你对"运动控制"**最根本的假设**——大脑真的是"中央指挥官"吗？有序行为是不是可以从身体+环境+任务的耦合中**涌现**出来？

### 👨‍🏫 核心人物

**J.A. Scott Kelso**（Florida Atlantic University, Center for Complex Systems & Brain Sciences）。虽然他不是 TAMU 的教授，但 KINE 642 半门课是围绕他的工作展开的。**John J. Buchanan** 是这门课最可能的授课教授（协调动力学专家）。

### 📝 核心概念笔记

#### 范式对比

```
信息处理范式：                自组织范式：
大脑 → 计算 → 执行 → 动作    身体+环境+任务 → 耦合 → 涌现 → 有序行为
（大脑是中央指挥官）          （不需要中央控制者）
```

#### HKB 模型（自组织的数学入门）

**经典实验**（Kelso, 1984）：两个食指以某个相对相位摆动，频率逐渐增加时：
- 低频：同相(in-phase)和反相(anti-phase)都稳定
- 频率到临界值：**反相突然跳到同相**（相变！）
- 频率降回来：反相**不会自动恢复**（滞后效应）

**核心概念速查**：

| 概念 | 定义 | 类比 |
|------|------|------|
| **序参量** | 描述系统宏观状态的变量 | 相对相位 φ |
| **控制参量** | 改变时导致相变 | 运动频率 |
| **吸引子** | 系统趋向的稳定状态 | 同相和反相是两个吸引子 |
| **相变** | 一个吸引子消失→跳到另一个 | 反相→同相的突然跳变 |
| **涨落增大** | 接近相变的变异性增大 | φ 的标准差变大 |
| **滞后** | 响应依赖于历史 | 降频率后反相不恢复 |

**HKB 势函数直觉**：

$$V(\phi) = -a \cos\phi - b \cos 2\phi$$

- 同相(φ=0°)的"谷"更深→更稳定
- 频率增加→反相(φ=180°)的"谷"变浅→消失→系统跳到同相

#### 学习策略

1. **从 HKB 模型入手**：这是全领域的数学基础。在 Python 中画出不同频率下的 V(φ) 曲线，直观感受吸引子如何消失
2. **不要试图"选边站"**：OFC 和 DST 在不同层次解释运动。OFC 解释单次运动的最优控制，DST 解释运动模式如何从耦合中涌现
3. **理解"Meta-theory"**：自组织是一种科学哲学立场，不只是运动控制理论
