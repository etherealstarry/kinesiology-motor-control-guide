## 🎓 KINE 601：Proseminar in Kinesiology

### 课程性质

博士新生必修的**系内研讨课**（1学分），每周邀请 KNSM 系内外教授/访问学者做研究报告。

**核心价值**：
- 了解系内教授的研究方向（方便选导师和轮转）
- 学习如何做学术报告（你也要讲！）
- 建立学术社交网络
- 了解研究伦理和学术规范

### 典型内容

| 环节 | 内容 |
|------|------|
| 教授研究分享 | KNSM 各方向（Motor Neuroscience、Exercise Physiology、Sport Management）教授轮番介绍 |
| 学生报告 | 每位博士生需做一次研究汇报（或研究计划展示） |
| 学术规范培训 | IRB 申请流程、科研伦理、负责任研究行为（RCR） |
| 职业技能 | 如何写 grant proposal、怎么做 peer review、学术写作技巧 |

### 🎯 学习策略

!!! tip "KINE 601 生存指南"
    1. **提前了解演讲教授**：上课前 Google 一下这周演讲者的研究方向，读一篇他们的代表作——提问时能说出具体内容会留下深刻印象
    2. **你的演讲是 showtime**：这是你在全系面前展示自己的机会。如果你已经在做 rotation 研究，务必展示 preliminary data
    3. **记笔记 - 不只是内容**：记录每位教授的 presentation style、slide design、Q&A handling——这些都是你以后做报告时的参考模板
    4. **主动社交**：seminar 后通常有 coffee hour / reception，别跑，这是建立人脉的关键时间
    5. **RCR 认证认真做**：CITI Program 培训虽然枯燥，但 IRB 申请时用得上

## 🧠 KINE 606：Motor Neuroscience I

### 课程性质

**Motor Neuroscience 第一门核心理论课**，系统建立运动控制与运动学习的神经科学框架。

> 已有完整课程笔记：[Motor-Neuroscience-I.md](必修课程/核心课程/Motor-Neuroscience-I.md)

### 核心主题

```
运动控制系统层次结构
├── 脊髓：CPG + 反射回路
├── 皮层下：小脑（内部模型/时序预测）、基底节（动作选择/强化学习）
├── 皮层：M1/PMd/PMv/SMA（运动计划与执行）
└── 前额叶：决策、目标设定

核心理论模型
├── 内部模型（正向模型 vs 逆向模型）
├── 最优反馈控制（OFC）
├── 贝叶斯感知模型
└── 运动学习的适应性 vs 获得性机制
```

### 👨‍🏫 授课教授

KINE 606 由 Motor Neuroscience 方向核心教授轮换授课。TAMU KNSM Motor Neuroscience 方向教授包括：

| 教授 | 研究方向 | 实验室 |
|------|----------|--------|
| (待确认) | 运动控制、运动学习 | Motor Behavior Lab |

> 入学后根据每年课程安排确认，或直接查 [KNSM Faculty](https://knsm.tamu.edu/faculty/) Motor Neuroscience 分组。

### 🎯 学习策略

!!! tip "博士级别：超越死记硬背"
    KINE 606 不是本科生的"记住皮层分区"——你需要达到**能用计算模型解释实验发现**的水平。

    1. **学一个概念 = 读一篇经典 + 一篇前沿**：
       - 经典：Wolpert, Diedrichsen & Flanagan (2011) → 内部模型框架
       - 前沿：查近3年的 *Nature Neuroscience* / *Neuron* / *eLife* 运动控制论文
    2. **用 Python 复现经典模型**：OFC 控制器、贝叶斯感知模型、强化学习 agent——代码比公式好理解
    3. **建立跨课程连接**：
       - KINE 606 的内部模型 → KINE 642 的动态系统 → 理解运动控制的不同理论视角
       - KINE 606 的可塑性 → KINE 640 的 TMS 研究 → 理解因果推断方法

## 🔬 KINE 640：Motor Neuroscience II

### 课程性质

KINE 606 的进阶课程，深入高级理论、神经调控技术与计算精神病学视角。

> 已有完整课程笔记：[Motor-Neuroscience-II.md](必修课程/核心课程/Motor-Neuroscience-II.md)

### 核心主题

| 模块 | 内容 |
|------|------|
| **高级理论对比** | FBS（平衡点假说）vs OFC（最优反馈控制）vs Active Inference（主动推理） |
| **神经调控技术** | TMS（经颅磁刺激）、tDCS/tACS（经颅电刺激）、fMRI、EEG |
| **高级学习机制** | 迁移学习、睡眠依赖巩固、元学习（meta-learning） |
| **社会运动神经科学** | 联合行动（joint action）、动作观察网络（AON）、镜像神经元 |
| **计算精神病学** | PD、中风、肌张力障碍的计算模型+行为实验 |

### 🎯 学习策略

!!! tip "KINE 640 的独特挑战"
    这门课从"学知识"进入"做研究"。你需要：
    
    1. **掌握至少一种神经调控技术**：
       - 如果实验室有 TMS：学 MEP 测量、TMS 运动映射、rTMS 协议
       - 如果实验室做 fMRI：学 MVPA（多体素模式分析）、RSA（表征相似性分析）
       - 通用：EEG 的 ERP + 时频分析（MNE-Python）
    2. **理论辩论能力**：FBS vs OFC vs Active Inference 的优劣——你需要能在 seminar 上为某个立场辩护
    3. **实验设计能力**：从"读完论文"转向"设计方案"。期中/期末通常要求写 research proposal

## 👶 KINE 641：Developmental Motor Neuroscience

### 课程性质

从**发展视角**研究运动控制与运动学习，覆盖**全生命周期**的运动行为变化。

> 已有完整课程笔记：[Motor-Neuroscience-Developmental.md](必修课程/核心课程/Motor-Neuroscience-Developmental.md)

### 核心主题

```
生命周期视角
├── 胎儿期：自发性运动的起源
├── 婴儿期：反射 → 自主运动（0-2岁）
├── 儿童期：精细/大肌肉动作发展（2-12岁）
├── 青春期：运动表现的巅峰
├── 成年期：运动技能的维持与优化
└── 老年期：运动退化（肌少症、平衡↓）

核心理论
├── Dynamic Systems Theory (Thelen, Smith)
├── Neuronal Group Selection Theory (Edelman)
├── 运动发育的约束模型
└── 关键期/敏感期假说
```

### 🎯 学习策略

!!! tip "发展视角的独特价值"
    运动控制的**发展视角**是最自然的实验——大脑和身体在发育过程中的变化本身就是"实验操作"。用它来测试你的理论是否robust。

    1. **从婴儿数据理解成人理论**：婴儿的运动学习模式常揭示成人研究中忽略的机制
    2. **纵向设计思维**：发展研究必然涉及时间维度，学会用 mixed-effects model / growth curve analysis
    3. **跨课程连接**：
       - KINE 641 的 Dynamic Systems Theory → KINE 642 的 Self-Organization → 它们是同一理论的两个视角
       - KINE 641 的发育约束 → KINE 606 的 Newell 约束模型 → 横贯全部课程

## 🔄 KINE 642：Self-Organization in Movement

### 课程性质

从**复杂系统/动态系统**视角理解运动控制——颠覆传统的"大脑是中央指挥官"观点。

> 已有完整课程笔记：[Self-Organization.md](必修课程/核心课程/Self-Organization.md)

### 核心理念：一场范式革命

```
传统观点（信息处理范式）：
  大脑 → 计算轨迹 → 发送指令 → 身体执行
  （大脑是"中央指挥官"）

动态系统观点（自组织范式）：
  身体 + 环境 + 任务 → 协同动力学 → 行为涌现
  （运动模式是多个子系统相互作用的"涌现"结果）
```

### 核心概念

| 概念 | 说明 | 经典实验 |
|------|------|----------|
| **协同动力学 (Coordination Dynamics)** | 多肢体协调的动力学规律 | Kelso 的手指摆动实验（同相→反相自发转换） |
| **序参量 (Order Parameter)** | 描述系统宏观状态的变量 | 相对相位（relative phase） |
| **控制参量 (Control Parameter)** | 改变导致系统状态变化的变量 | 运动频率（frequency） |
| **相变 (Phase Transition)** | 系统从一个稳定状态跳到另一个 | 频率增加时，同相协调突然变为反相 |
| **吸引子 (Attractor)** | 系统趋向的稳定状态 | 同相协调和反相协调是两个吸引子 |
| **涨落 (Fluctuation)** | 相变前变异性增大 | 频率接近临界值时，相对相位的标准差增大 |

### 👨‍🏫 核心人物：J.A. Scott Kelso

| 信息 | 说明 |
|------|------|
| **身份** | Florida Atlantic University, Center for Complex Systems & Brain Sciences |
| **代表作** | *Dynamic Patterns: The Self-Organization of Brain and Behavior* (MIT Press, 1995) |
| **核心贡献** | HKB 模型（Haken-Kelso-Bunz）、协同动力学框架 |
| **学术传承** | Kelso 的学生遍布全球，形成"Kelso学派" |

### 🎯 学习策略

!!! tip "自组织视角的核心：从"控制"到"涌现""
    
    KINE 642 可能和你在其他课程学到的内容冲突——这是正常的。

    1. **拥抱跨范式思维**：不要试图"选择"一个理论。OFC 告诉我们大脑如何优化控制，DST 告诉我们很多行为不需要大脑优化就能涌现。两个都对，取决于你在哪个层次解释。
    2. **HKB 模型是入门钥匙**：先彻底理解 HKB 模型的数学（两个耦合振子的微分方程），这是整个领域的数学基础。
    3. **Meta-theory 视角**：自组织不是"运动控制理论"，而是一种**科学哲学立场**——任何复杂系统（神经、肌肉、环境耦合）都可能产生有序行为，不需要中央控制者。
    4. **跨课程连接**：
       - KINE 642 的自组织 → KINE 641 的发展动态系统 → Thelen 用 DST 解释婴儿行走的出现
       - KINE 642 的吸引子 → KINE 606 的运动学习 → 学习 = 吸引子景观的改变
