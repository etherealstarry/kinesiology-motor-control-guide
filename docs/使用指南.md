## 🚀 快速开始（New Studens Start Here）

### 第 1 步：了解培养方案

访问 **[学习规划](./学习规划.md)**，查看按学年排列的推荐课程顺序。

TAMU Motor Neuroscience PhD 核心课程号：
- `KINE 601` — Proseminar（博士必修研讨）
- `KINE 606` — Motor Neuroscience I（**核心核心！**）
- `KINE 640` — Motor Neuroscience II
- `KINE 641` — Developmental Motor Neuroscience
- `KINE 642` — Self-Organization in Movement

### 第 2 步：选统计课

访问 **[统计工具](./必修课程/统计工具/统计课程.md)**，选择你的第一门统计课。

**最推荐**：`STAT 652 — Statistics for Experimenters I`（实验科学首选）

### 第 3 步： setup 文献管理

访问 **[文献管理](./学习工具/文献管理.md)**，安装 Zotero 并学会基本操作。

> 💡 这条省下的时间会以**百小时**计！

### 第 4 步：学 R 或 Python

访问 **[数据分析工具](./学习工具/数据分析工具.md)**，选择并学习一门数据分析工具。

> 💡 R 更适合统计分析，Python 更适合计算建模。建议两门都学，第 1 年主攻 R。

## 📋 如何使用本指南做课程规划

### 对照 TAMU 课程目录

1. 访问 [TAMU Course Catalog](https://catalog.tamu.edu/)
2. 搜索 `KINE`、`NRSC`、`STAT`、`EDCI` 前缀的课程
3. 对照本指南的「学习规划」页面，检查课程是否匹配
4. 与导师讨论选课计划

### 每学期选课建议

!!! tip "来自学长的建议"
    ```
    每学期 9 credits（全职学生最低要求）：
    ├── 2 门学术课程（6 credits）
    ├── 1 门选修或 KINE 691（3 credits）
    └── 如有 TA 职责，相应减少课程
    
    注意：KINE 691（Research）是 S/U 评分，不影响 GPA！
    ```

## 📝 做笔记的建议

### 方法 1：打印 + 手写标注

- 把关键页面（如统计课程、研究方向）打印出来
- 手写补充你自己的经验和笔记
- 建立自己的「纸质参考书」

### 方法 2：Obsidian / Notion 数字化

- 用 Obsidian 做双向链接笔记
- 每篇论文一个笔记，相互链接
- 与 Zotero 联动（Zotero Integration 插件）

### 方法 3：Fork 本仓库，直接编辑

如果你想直接修改本网站内容：

1. Fork [etherealstarry/kinesiology-motor-control-guide](https://github.com/etherealstarry/kinesiology-motor-control-guide)
2. 在本地克隆你的 Fork
3. 修改 Markdown 文件
4. 提交 Pull Request（或直接 push 到你的 Fork）
5. 你的修改会出现在本指南的未来版本中！
