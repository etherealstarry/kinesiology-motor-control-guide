# 如何使用这个指南

本指南旨在帮助你系统性地学习运动控制（Motor Control）相关知识，无论你是刚入门的新生，还是希望深入某个方向的研究者，都可以从中找到适合自己的学习路径。

## 📖 本指南的组织结构

本指南按照以下逻辑组织内容：

### 1. 基础课程
涵盖运动机能学的基础理论知识，包括：
- 解剖学基础
- 生理学基础  
- 运动机能学基础

### 2. 核心课程
运动控制方向的核心专业课程，包括：
- 运动控制理论
- 运动学习
- 神经力学
- 生物力学
- 运动发育

### 3. 研究方法
科研必备的方法学训练，包括：
- 实验设计与统计
- 科研方法

### 4. 研究方向
运动控制领域的主要研究方向介绍：
- 运动控制与学习
- 神经康复
- 生物力学应用
- 运动表现优化

### 5. 学习工具
学习和科研中常用的工具：
- 文献管理
- 数据分析工具
- 实验设备使用
- 学术写作

### 6. 拓展资源
超出课程之外的学习资源：
- 在线课程
- 学术会议
- 研究实验室
- 实习与就业

## 🎯 如何使用本指南

### 如果你是...

**🎓 本科生（第1-2年）**
1. 从「基础课程」开始，打好理论基础
2. 按照课程顺序学习，不要跳跃
3. 每门课程都要完成推荐的阅读材料
4. 尽早开始参与实验室研究

**🎓 本科生（第3-4年）**
1. 完成核心课程学习
2. 选择1-2个研究方向深入了解
3. 学习科研方法和数据分析工具
4. 开始进行独立研究项目

**🎓 研究生**
1. 根据研究方向选择深入课程
2. 重点关注研究方法和前沿进展
3. 学习专业的数据分析工具
4. 广泛阅读相关领域文献

**🔬 研究者**
1. 根据研究需要选择性学习
2. 重点关注拓展资源和前沿进展
3. 参考好书推荐进行深入理解

## 📋 学习建议

### 1. 理论与实践结合
- 不要只读书，要动手实践
- 尽早进入实验室参与研究
- 尝试复现经典研究

### 2. 建立知识体系
- 做好笔记，建立知识之间的联系
- 定期复习，巩固理解
- 绘制概念地图，理清逻辑关系

### 3. 积极参与学术社区
- 参加学术会议和研讨会
- 加入相关学术组织
- 与同行交流讨论

### 4. 培养批判性思维
- 不要盲目接受文献结论
- 学会评估研究质量
- 思考研究的局限性和未来方向

## 🤝 贡献指南

本指南是开源项目，欢迎贡献！

### 如何贡献

1. **新增课程资源**
   - 在对应目录下创建新的课程文档
   - 使用提供的模板格式（见下方）
   - 在 `mkdocs.yml` 中添加导航链接

2. **推荐好书**
   - 在 `好书推荐.md` 中按照格式添加
   - 说明推荐理由和适用人群

3. **修正错误**
   - 发现错误直接在GitHub提交Issue
   - 或者提交Pull Request进行修正
