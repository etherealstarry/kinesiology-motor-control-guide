|------|------|

---

## 🔬 核心知识体系

| 模块 | 内容 |
|------|------|
| **神经肌肉适应** | 运动单位募集、同步化、发射频率、交叉迁移 |
| **肌肉肥大 (Hypertrophy)** | 肌原纤维 vs 肌质肥大、mTOR 通路、机械张力/代谢应激/肌肉损伤 |
| **力量-速度关系** | Force-Velocity Curve、功率峰值、训练特异性 |
| **周期化 (Periodization)** | 线性、波动 (DUP)、板块 (Block)、共轭 (Conjugate) |
| **疲劳机制** | 中枢疲劳 vs 外周疲劳、代谢物积累、Ca²⁺ 处理 |
| **爆发力训练** | 弹道训练、PAP/Complex Training、举重衍生动作 |
| **恢复与适应** | 超量恢复、功能性与非功能性过度、overtraining 综合征 |

## 📝 自编概念闪卡（20 张）

| # | 正面 | 背面 |
|---|------|------|
| 1 | Henneman's Size Principle？ | 小运动神经元先募集，大运动神经元后募集 |
| 2 | 力量增长的神经适应期 vs 肌肉适应期？ | 前 4-6 周=神经适应（IRC↑）；之后=肌肉肥大 |
| 3 | 肌原纤维肥大 vs 肌质肥大？ | 肌原纤维=收缩蛋白↑；肌质=糖原/肌浆↑（健美式"泵感"） |
| 4 | 机械张力是肥大的主要驱动？ | 是的——mTOR 通路对机械负荷敏感 |
| 5 | 训练到力竭 (Failure) 对肥大的必要性？ | 非必要——接近力竭即可，力竭增加疲劳而不显著增加肥大 |
| 6 | 线性周期化 (LP) 的特点？ | 容量↓ 强度↑——经典但证据支持弱于波动周期化 |
| 7 | 每日波动周期化 (DUP)？ | 每天变化训练刺激（如周一力量/周三肥大/周五爆发） |
| 8 | 板块周期化 (Block)？ | 2-4 周集中发展一个能力→串联累积 |
| 9 | Force-Velocity Curve 的"甜蜜点"？ | 最大功率 Pmax≈30-40% Vmax |
| 10 | PAP (Post-Activation Potentiation) 机制？ | 重负荷→肌球蛋白轻链磷酸化→增强后续爆发力 |
| 11 | Complex Training 的典型方案？ | 重 squat (85%1RM×3) → 休息 3-8min → box jump |
| 12 | RFD (Rate of Force Development) 的定义？ | 力的发展速率——△Force/△Time—爆发力的关键指标 |
| 13 | DOMS 的机制？ | Delayed Onset Muscle Soreness——微损伤+炎症反应 |
| 14 | Overtraining 综合征的标记？ | 持续表现下降+情绪紊乱+静息 HR↑+HRV↓ |
| 15 | 1RM 测试的黄金标准？ | 标准化 warmup + 5-6 次递增 + 尝试→成功→休息3-5min |
| 16 | Sets per week 对肥大的剂量效应？ | ~10-20 sets/muscle/week 获得最大肥大 |
| 17 | Eccentric 训练的特殊性？ | 更大力量产出、更多肌肉损伤、3-4 天恢复期 |
| 18 | Velocity-Based Training (VBT) 的优势？ | 实时反馈、自动调节、不需要频繁测试 1RM |
| 19 | 女性 vs 男性力量训练的核心差异？ | 相同相对增益、不同绝对力量——肌肉质量差异 |
| 20 | 老年人力量训练的特殊考虑？ | 安全第一、高频率低强度、重在功能而非负重 |

## ✏️ 自编模拟考题

**Q1**：设计一个 12 周的周期化训练方案，目标是增加深蹲 1RM。包含：周期化类型、每周结构、sets/reps/强度。

**Q2**：比较"A 组训练到力竭 vs B 组留 2 次余力"对肌肉肥大的影响。预期结果是什么？为什么？

**Q3**：解释 PAP 的生理机制，并设计一个 Complex Training 方案（举重 + 跳跃）。
