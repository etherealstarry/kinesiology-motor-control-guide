## 🔬 核心知识体系

### 范式革命

```
传统信息处理范式              动态系统/自组织范式
─────────────────          ─────────────────
大脑 → 计划 → 执行 → 动作    身体+环境+任务 → 耦合 → 涌现 → 行为
（Central Controller）        （No Central Controller）

核心差异：
✗ 不是"大脑不重要"
✓ 而是"大脑只是复杂系统中的一个子系统"
```

## 📝 自编概念闪卡

### HKB 模型（15 张）

| # | 正面 | 背面 |
|---|------|------|
| 1 | HKB 实验 (Kelso, 1984) 的经典范式？ | 两个食指以相对相位摆动，频率逐渐增加→反相突然跳到同相（相变！） |
| 2 | HKB 势函数的公式？ | V(φ) = -a·cos(φ) - b·cos(2φ) |
| 3 | a 和 b 在 HKB 方程中的含义？ | b/a 决定耦合强度——频率增加时 b/a 减小，反相吸引子变浅 |
| 4 | 序参量 (Order Parameter) 的定义？ | 描述系统宏观状态的变量——在 HKB 中是相对相位 φ |
| 5 | 控制参量 (Control Parameter) 的定义？ | 改变时导致相变的变量——频率 |
| 6 | 相变 (Phase Transition) 的标志？ | 序参量的突然跳变 + 涨落增大 + 临界减速 |
| 7 | 涨落增大 (Critical Fluctuations) 的含义？ | 接近相变时 φ 的标准差增大 — 系统在吸引子间"犹豫" |
| 8 | 滞后 (Hysteresis) 的现象？ | 降频率后反相不自动恢复 — 系统状态依赖于历史路径 |
| 9 | 多稳态 (Multistability)？ | 同一控制参量值可以对应多个稳态 — 取决于初始条件 |
| 10 | 同相 (In-phase) 为什么比反相 (Anti-phase) 更稳定？ | 同相的势阱更"深"（谷更深）——V(0°) < V(180°) |

### 自组织核心概念（15 张）

| # | 正面 | 背面 |
|---|------|------|
| 11 | 自组织 (Self-Organization) 的定义？ | 系统在没有中央控制者的情况下，从组件间局部相互作用中自发产生有序模式 |
| 12 | 涌现 (Emergence) 的定义？ | 系统整体的行为不能从组件的性质直接预测 — "整体大于部分之和" |
| 13 | Bernstein 的自由度数问题？ | 人体有 ~100 个机械自由度 — CNS 不可能单独控制每个 — 通过"协同"减少自由度 |
| 14 | 协同 (Synergy / Coordinative Structure) 的定义？ | 一组肌肉/关节被临时"绑定"在一起作为一个功能单位运作 |
| 15 | 吸引子 (Attractor) 是什么？ | 系统趋向的稳定状态 — 如同碗底的球，扰动后自动回到碗底 |
| 16 | 感知-动作耦合 (Perception-Action Coupling)？ | 感知和动作不是先后关系而是循环因果关系 — 感知指导动作，动作产生新的感知 |
| 17 | Intrinsic Dynamics vs Behavioral Information？ | 内在动力学（系统已有倾向）+ 行为信息（任务要求）→ 实际行为模式 |
| 18 | 稳定性 (Stability) 的测量方法？ | 局域动态稳定性：Lyapunov 指数；变异性：SD of φ |
| 19 | 复杂系统 ≠ 复杂行为？ | 简单规则 → 复杂模式（如鸟群飞行、集体同步） |
| 20 | Schöner-Kelso 协同动力学？ | 用微分方程描述协同模式的形成、稳定和切换 |

### 方法论（10 张）

| # | 正面 | 背面 |
|---|------|------|
| 21 | 相对相位 (Relative Phase) 的计算？ | φ = θ₁ - θ₂（两个节律运动之间的相位差） |
| 22 | 循环图 (Cyclic Graph) 的作用？ | 展示两个变量之间的相位关系——逆时针圆=反相，对角线=同相 |
| 23 | Hilbert 变换在运动分析中的作用？ | 从时间序列提取瞬时相位 |
| 24 | 递归量化分析 (RQA) 是什么？ | 分析动力学系统的递归模式——识别确定性和随机性 |
| 25 | Detrended Fluctuation Analysis (DFA)？ | 量化时间序列的长程相关性（如步态中的 fractal 特性） |

## 🐍 Python 练习（OFC vs DST 对比理解）

```python
# HKB 势函数可视化
import numpy as np
import matplotlib.pyplot as plt

phi = np.linspace(-np.pi, np.pi, 500)
a = 1.0

ratios = [0.5, 0.3, 0.2, 0.15]
for b_a in ratios:
    b = b_a
    V = -a * np.cos(phi) - b * np.cos(2*phi)
    plt.plot(phi, V, label=f'b/a={b_a}')

plt.xlabel('Relative Phase φ (rad)')
plt.ylabel('Potential V(φ)')
plt.legend()
plt.title('HKB Model: Potential as coupling strength decreases')
plt.grid(True)
plt.show()
```

## 🔥 高分策略

### Buchanan 的招牌课 — 这是他毕生研究

1. **尊重这门课** — 这是他 40 年研究的结晶，你必须深入理解而非表面背诵
2. **不要试图"站队"** — OFC 和 DST 在不同层次解释运动，两者不是互斥的
3. **数学是理解的门槛** — HKB 势函数、分叉图、相空间图必须动手画/编程
4. **他的问题会逼你到墙角** — 准备充分，但承认"我不知道"并尝试推理也比胡说好
5. **如果你对 DST 真的感兴趣** — 这门课可能决定你的研究方向（Buchanan 需要博士生）
6. **Final paper：DST 框架下的论文 proposal** — 选一个你感兴趣的行为现象，用自组织框架设计实验

### 常见误区

| ❌ 错误 | ✅ 正确 |
|--------|--------|
| "大脑在自组织中没有作用" | 大脑是系统的一部分，提供"行为信息"(behavioral information) |
| "OFC 和 DST 是对立的" | 不同分析层次——OFC 解释单次动作的最优化，DST 解释模式的涌现 |
| "HKB 模型只是数学游戏" | HKB 预测了实际实验中发现的相变现象，是预测性的数学模型 |
| "背公式就能过" | Buchanan 的课不考公式背诵——考你是否理解"为什么反相不稳定" |
