## 🔬 核心知识体系

### Grant Writing 全流程

```
找机会 (FOA/RFA) → Specific Aims → Significance → Innovation → Approach → 预算 → 提交
```

| 模块 | 内容 |
|------|------|
| **基金类型** | NIH R01/R21/R03/F31/K99、NSF、基金会 |
| **Specific Aims** | 1 页纸说清楚：要解决什么问题、为什么重要、怎么做 |
| **Significance** | 文献综述 + 知识缺口 + 你的研究填补什么 |
| **Innovation** | 方法创新？概念创新？技术创新？ |
| **Approach** | 实验设计、方法、分析计划、备选方案、时间线 |
| **预算与 justification** | 人员、设备、耗材、差旅、间接成本 |
| **评审过程** | Study Section、评分标准、Resubmission 策略 |

## 📝 自编概念闪卡（15 张）

| # | 正面 | 背面 |
|---|------|------|
| 1 | NIH R01 是什么？ | 研究者发起的研究项目基金，5 年，~$250K/年 |
| 2 | R21 vs R03？ | R21=探索性(2年$275K)；R03=小型研究(2年$100K) |
| 3 | F31 是什么？ | NIH 博士前国家研究服务奖（博士生的 fellowship） |
| 4 | K99/R00 是什么？ | 博士后→独立研究者过渡基金 |
| 5 | Specific Aims 页的黄金公式？ | 1 段背景 + 2-3 个 aims（每个 1 段）+ 预期结果 + 影响 |
| 6 | Significance 要回答什么问题？ | 如果不做这个研究，field 会失去什么？ |
| 7 | Innovation 不是说什么？ | 不是说"我们是第一个做这个的"——要说方法/概念的新颖性 |
| 8 | Approach 的 Preliminary Data 为什么重要？ | 证明你有能力做这个项目、假设有基础 |
| 9 | Impact Score 的范围？ | 1（最佳）到 9（最差） |
| 10 | Payline 是什么？ | 能获得资助的 Impact Score 阈值 |
| 11 | Modular Budget vs Detailed Budget？ | Modular=按 $25K 模块化；Detailed=逐项明细 |
| 12 | Resubmission 策略？ | 认真读 Summary Statement → 逐条回复 reviewer concerns |
| 13 | Biosketch 包含什么？ | 教育、职位、荣誉、contributions to science、个人声明 |
| 14 | FOA vs RFA 的区别？ | FOA=一般性资助公告；RFA=特定主题征集 |
| 15 | 间接成本 (Indirect Costs) 是什么？ | Facilities & Administrative 成本，通常是直接成本的 ~50%+ |

## ✏️ 自编模拟考题

**Q1**：为你的博士研究方向写一页 Specific Aims。包含：背景、知识缺口、2 个 aims、预期结果。

**Q2**：你收到 Reviewer 的 critique："Aim 2 的实验设计不足以检验假设。"请写一段 Resubmission 的回复。

**Q3**：比较 NIH R21 和 NSF EAGER 在探索性研究方面的资助哲学差异。

## 🔥 学习策略

1. **这门课的 final paper = 你的真实 grant proposal** — 一稿两用，省时间
2. **提前想好一个具体的研究问题** — 整个学期都在写这个 grant
3. **去 TAMU SRS 找 Grant Specialist 看你的 budget** — 免费资源不用白不用
4. **读 3-5 篇已资助的 Sample Grant** — 学习成功者的叙事结构
