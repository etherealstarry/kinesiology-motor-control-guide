## 🔬 核心知识体系

### 研究方法全流程

```
研究问题 → 文献综述 → 假设 → 实验设计 → 数据收集 → 分析 → 解释 → 发表
```

| 模块 | 核心内容 |
|------|----------|
| **科学哲学** | 实证主义 vs 诠释主义、假设检验、证伪主义 (Popper) |
| **实验设计** | 真实验 vs 准实验、内部效度 vs 外部效度、随机化 |
| **测量** | 信度 (reliability)、效度 (validity)、操作化定义 |
| **抽样** | 概率抽样 vs 非概率抽样、样本量计算 (Power Analysis) |
| **伦理** | IRB、知情同意、数据管理、科研诚信 |

## 📝 自编概念闪卡

### 研究方法核心概念（20 张）

| # | 正面 | 背面 |
|---|------|------|
| 1 | 内部效度 vs 外部效度？ | 内部=因果推断的可信度；外部=结果的可推广性 |
| 2 | 随机分配 vs 随机抽样？ | 分配=谁进哪组；抽样=谁参与研究 |
| 3 | Type I Error vs Type II Error？ | I=假阳性(拒真)；II=假阴性(存伪) |
| 4 | 统计功效 (Power) 的定义？ | 当效应真实存在时，正确拒绝零假设的概率 = 1-β |
| 5 | Power Analysis 的四个参数？ | α, Power(1-β), Effect Size, N——已知三个求第四个 |
| 6 | 内部一致性信度 (Cronbach's α)？ | 测量工具各条目之间的一致性程度 |
| 7 | 重测信度 (Test-retest Reliability)？ | 同一测量在不同时间的一致性 |
| 8 | 内容效度 vs 构念效度？ | 内容=覆盖全部维度；构念=测量到了理论构念 |
| 9 | 混淆变量 (Confounding Variable)？ | 与研究变量相关、影响因变量、但未控制的变量 |
| 10 | 中介变量 vs 调节变量？ | 中介=解释"怎么影响"；调节=影响"什么时候/对谁影响" |
| 11 | 纵向设计 vs 横断设计的优劣？ | 纵向=看变化但耗时长；横断=快但有队列效应 |
| 12 | RCT 的金标准地位？ | 随机分配控制混淆变量→最强的因果推断 |
| 13 | 单盲 vs 双盲 vs 三盲？ | 单=被试不知；双=被试+实验者不知；三=被试+实验者+分析者 |
| 14 | 效应量 (Cohen's d / η²) 为什么重要？ | p 值只告诉你"有差异"，效应量告诉你"差异多大" |
| 15 | 混合方法研究 (Mixed Methods)？ | 定量 + 定性方法结合，互补优势 |

## ✏️ 自编模拟考题

**Q1**：你想研究"HIIT 训练对反应时间的影响"。写出你的研究问题、IV、DV、实验设计类型、可能的混淆变量、和如何控制它们。

**Q2**：一个研究发现"每天冥想 10 分钟可以降低血压 (p=0.04, d=0.15)"。p 显著但 d 很小。这个结果有什么实际意义？你会怎么报告它？

**Q3**：你要设计一个问卷测量"运动动机"。请列出建立内容效度和构念效度的步骤。

## 🔥 学习策略

1. **这门课是 Prelim 方法论部分的核心** — 考试会问你"如何设计一个研究回答 X 问题"
2. **研究方法不是"理论"而是"工具"** — 边学边用在你自己的研究 proposal 里
3. **Power Analysis 必须会算** — 下载 G*Power 软件练习
4. **效度和信度是高频考点** — 能脱口说出每种的定义和检验方法
