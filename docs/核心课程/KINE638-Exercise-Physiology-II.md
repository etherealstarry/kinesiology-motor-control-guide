## 🔬 核心知识体系

| 模块 | 内容 |
|------|------|
| **心血管调控** | 心率、每搏输出量、心输出量、血压调控、VO₂max |
| **呼吸调控** | 通气量、气体交换、氧解离曲线、通气阈值 |
| **内分泌响应** | 儿茶酚胺、皮质醇、GH、IGF-1、胰岛素/胰高血糖素 |
| **代谢整合** | 磷酸原→糖酵解→氧化磷酸化、底物利用 (RER) |
| **体温调节** | 蒸发/对流/辐射/传导、热应激和冷应激 |
| **高原适应** | 低氧→EPO↑→Hct↑、通气↑、毛细血管密度↑ |
| **训练适应** | 线粒体生物发生 (PGC-1α)、毛细血管化、心脏重塑 |

## 📝 自编概念闪卡（20 张）

| # | 正面 | 背面 |
|---|------|------|
| 1 | VO₂max 的定义和公式？ | 最大摄氧量 = Q_max × a-vO₂diff_max |
| 2 | VO₂max 的中央限制 vs 外周限制？ | 中央=心输出量；外周=肌肉氧提取和利用 |
| 3 | 220-age 公式的准确度？ | ±10-12 bpm——仅作粗略估计 |
| 4 | Karvonen 公式（储备心率法）？ | THR = (HRmax - HRrest) × %intensity + HRrest |
| 5 | 通气阈值 (VT1/VT2) 的生理意义？ | VT1=乳酸开始积累；VT2=呼吸代偿点 |
| 6 | RER (呼吸交换率) 的含义？ | VCO₂/VO₂——0.7=脂肪氧化；1.0=碳水氧化；>1.0=无氧代谢 |
| 7 | EPO 的作用机制？ | 肾(低氧)→EPO 分泌→骨髓红细胞生成→Hct↑→携氧量↑ |
| 8 | 热适应的核心变化？ | 出汗率↑、皮肤血流↑、血浆量↑、Na⁺重吸收↑ |
| 9 | PGC-1α 在耐力训练适应中的角色？ | 线粒体生物发生的"主开关" |
| 10 | HIIT 和中等强度持续训练的分子信号差异？ | HIIT→AMPK/PGC-1α 快速激活；两者均促进线粒体适应 |
| 11 | 运动员心脏的特征？ | 左室壁增厚(向心性) + 腔室增大(离心性)—取决于运动类型 |
| 12 | 运动强度对底物利用的影响（Crossover Concept）？ | 低强度=脂肪为主；高强度=碳水为主 |
| 13 | 乳酸不是废物——它的角色？ | 重要的糖异生前体 + 细胞间能量穿梭 |
| 14 | Fick Equation？ | VO₂ = Q × (CaO₂ - CvO₂)—摄氧量=心输出量×动静脉氧差 |
| 15 | 每搏输出量的 Frank-Starling 机制？ | 静脉回流↑→心室舒张末容积↑→收缩力↑→SV↑ |
| 16 | 运动后过量氧耗 (EPOC) 的组成？ | 快速=PCr 补充+O₂储备；慢速=乳酸清除+体温↑+HR↑ |
| 17 | 海拔对 VO₂max 的影响？ | 每 1000m→VO₂max↓~6-8% |
| 18 | 闭气训练 (BH) 的潜在益处？ | EPO↑、脾收缩→循环红细胞↑ |
| 19 | 女性的 VO₂max 为什么普遍低于男性？ | 体脂率↑、Hb↓、SV↓——相对值差异缩小 |
| 20 | Detraining 的时间进程？ | 2 周→VO₂max↓开始；4 周→显著↓；8 周→回到基线 |

## ✏️ 自编模拟考题

**Q1**：一个耐力运动员在海拔 2500m 训练 3 周后 VO₂max 下降了 15%。是"适应失败"吗？解释高原适应的生理变化时间线。

**Q2**：绘制运动强度增加时 RER 的变化曲线，标注"crossover point"。用底物代谢图谱解释。

**Q3**：比较 HIIT (4×4min @90% HRmax) 和 MICT (45min @65% HRmax) 在 8 周后的生理适应差异。
