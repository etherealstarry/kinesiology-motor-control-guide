# KINE 682: Seminar in...

> 专题研讨 | 1-3 学分 | 每学期

---

## 📋 课程信息

| 维度 | 详情 |
|------|------|
| 学分 | 1-3 |
| 学期 | 按需开设 |
| 形式 | 围绕特定专题（如 Motor Neuroscience / Exercise Physiology）深入研讨 |
| 评估 | 论文 presentation + 讨论参与 |

---

## 📝 速查

| 要做的事 | 为什么 |
|----------|--------|
| 选与自己研究方向一致的 Seminar | 直接积累领域知识 |
| 每次讨论至少发言 1 次 | 博士课程的 value 在于讨论，不在于听 |
| 提前读 paper | 提前 1 天读 + 标注 2-3 个问题 |

---

## 🔥 策略

1. **这比 681 更聚焦** — 围绕一个专题深入，对 Prelim 考试有帮助
2. **Presentation 是核心考核** — 练习 15-20 分钟的学术报告，跟 KINE 601 学到的技巧结合
3. **如果和导师方向重合** — 可以和导师商量把 682 和你的研究 rotation 内容结合
