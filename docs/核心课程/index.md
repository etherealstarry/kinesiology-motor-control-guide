## 📖 课程资源对比

| 维度 | 本科课 (406/426) | 博士课 (606/640/641/642) |
|------|-----------------|------------------------|
| 班级规模 | 30-60 人 | 8-15 人 |
| Quizlet/Stuvia | ✅ 丰富 | ❌ 几乎没有 |
| 考试形式 | 选择题 + 填空 + 简答 | 论文讨论 + presentation + final paper |
| 备考方式 | 刷闪卡 + 做往届题 | 读论文 + 批判讨论 + 研究 proposal |
| 你的角色 | 资源使用者 | **资源创造者**（你的笔记就是下届资源） |
