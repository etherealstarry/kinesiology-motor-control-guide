## 🔬 核心知识体系

| 模块 | 仪器/技术 | 测量内容 |
|------|-----------|----------|
| **心肺测量** | 代谢车 (Metabolic Cart) | VO₂、VCO₂、RER、通气量 |
| **心电图 (ECG)** | 12 导联 ECG、Holter | HR、HRV、ST 段分析 |
| **血压** | 血压计、Finger Cuff | SBP/DBP、MAP、BPV |
| **血乳酸** | Lactate Pro、YSI | 血乳酸浓度 (mM) |
| **肌氧** | NIRS (近红外光谱) | 组织氧饱和度、脱氧血红蛋白 |
| **体成分** | DXA、Bod Pod、BIA | 体脂率、瘦体重、骨密度 |
| **肌电 (EMG)** | 表面 EMG (sEMG) | 肌肉激活时间、幅度、疲劳 |
| **超声** | 肌骨超声 | 肌束长度、羽状角、肌腱力学 |
| **热成像** | 红外热像仪 | 皮肤温度分布 |

## 📝 自编概念闪卡（15 张）

| # | 正面 | 背面 |
|---|------|------|
| 1 | 代谢车测量 VO₂ 的原理？ | 收集呼出气→测量 O₂% 和 CO₂% + 通气量→VO₂=VI×FIO₂-VE×FEO₂ |
| 2 | NIRS 的原理？ | 近红外光穿透组织→氧合/脱氧 Hb 吸收不同波长→计算组织氧饱和度 |
| 3 | Lactate Pro 的检测原理？ | 酶电极法——乳酸氧化酶催化→H₂O₂→电化学检测 |
| 4 | DXA 的双能 X 光原理？ | 两种能量 X 光→软组织和骨骼吸收差异→区分脂肪/瘦体重/骨 |
| 5 | Bod Pod 的原理？ | 空气置换体积描记法——Boyle's Law (P×V=constant) |
| 6 | BIA (生物电阻抗) 的局限？ | 受水合状态显著影响——运动后出汗会大幅高估体脂 |
| 7 | HRV 的时域指标？ | SDNN（总变异）、RMSSD（迷走神经活性）、pNN50 |
| 8 | HRV 的频域指标？ | LF（交感+副交感）、HF（副交感/呼吸性窦性心律不齐）、LF/HF 比值 |
| 9 | EMG 信号的频率范围？ | 10-500 Hz（sEMG）— 需要滤波和整流 |
| 10 | ECG 的 P-QRS-T 波分别代表什么？ | P=心房去极；QRS=室去极；T=室复极 |
| 11 | 超声 B-Mode vs M-Mode 的区别？ | B-Mode=2D 解剖图像；M-Mode=1D 随时间变化（常测肌束） |
| 12 | 肌骨超声测量羽状角的步骤？ | 识别深/浅层腱膜→测肌束与深层腱膜夹角 |
| 13 | 红外热像仪的测量原理？ | 皮肤辐射红外能量→传感器转化为温度图像 |
| 14 | Finger Cuff 血压测量的原理？ | 体积钳技术 (Volume Clamp)——维持手指动脉恒定容积 |
| 15 | EMG 的 Normalization 为什么重要？ | 个体差异→绝对值无法比较→需用 MVC 归一化 |

## ✏️ 自编模拟考题

**Q1**：受试在代谢车上跑 3 个递增速度 (8/10/12 km/h)。VO₂ 从 2.0→2.8→3.5 L/min，RER 从 0.75→0.85→1.05。在每个速度下受试主要氧化什么底物？为什么 12 km/h 时 RER>1.0 不代表 RQ>1.0？

**Q2**：你给受试测了 DXA 和 BIA，体脂率分别为 25% 和 22%。为什么会有差异？哪个更可信？

**Q3**：设计一个使用 NIRS + sEMG 的实验，研究自行车递增负荷中肌肉氧合和激活的协同变化。

## 🔥 学习策略

1. **这门课的核心是"动手"** — 不是读 paper，是学会操作仪器
2. **每个仪器至少独立操作 3 次** — 第一次看、第二次做、第三次当 TA 教别人
3. **了解每个技术的原理和局限** — 不能只按按钮不思考
