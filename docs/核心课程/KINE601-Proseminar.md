# KINE 601: Proseminar in Kinesiology

> 博士新生必修 | 1 学分 | Fall 学期 | John J. Buchanan（Program Chair 协调）

---

## 📋 课程信息

| 维度 | 详情 |
|------|------|
| 学分 | 1（P/F 或 Letter Grade） |
| 学期 | Fall（第一学期） |
| 班级规模 | 全系新生（~15 人） |
| 授课方式 | 每周 1 次 Seminar，教授轮流主讲 + 学生报告 |
| 核心目标 | 从"学生"过渡到"学者" |

---

## 📑 每周内容与应对策略

| 阶段 | 内容 | 你的行动 |
|------|------|----------|
| **Week 1-2** | 项目介绍、博士时间线、系内规章 | 搞清楚 Prelim 时间线、学位要求、rotation 规则 |
| **Week 3-8** | 教授轮流分享研究方向 | 提前查背景 → 准备 1-2 个有深度的问题 |
| **Week 9-10** | RCR（负责任研究行为）+ IRB 伦理培训 | 完成 CITI Program 在线模块（课外） |
| **Week 11-13** | **学生报告**（每人 15-20 分钟） | 提前 2 周准备 slides → 试讲 2 次 → 严格计时 |
| **Week 14-15** | Grant writing workshop | 带上你的 proposal 草稿或研究想法 |

## 📝 自编概念闪卡

### 礼仪与规范

| 概念 | 定义 | 注意事项 |
|------|------|----------|
| RCR | Responsible Conduct of Research（负责任研究行为） | CITI Program 在线完成 |
| IRB | Institutional Review Board（机构审查委员会） | 人体实验必须通过 IRB 审批 |
| IACUC | 动物实验伦理审查 | 动物实验必须通过 |
| COI | Conflict of Interest（利益冲突） | 每年需要申报 |
| Authorship | 署名权 | 讨论：谁可以在论文署名？什么贡献才算？ |
| Mentoring | 导师关系 | 导师和学生之间的期望如何对齐 |

### 学术技能

| 概念 | 关键要素 |
|------|----------|
| **Elevator Pitch** | 30 秒说清楚你的研究兴趣 → 为什么重要 → 你想做什么 |
| **Academic Networking** | 会议社交、跟教授发冷邮件、Twitter/X 学术圈 |
| **Journal Club** | 15 分钟讲一篇论文：背景 → 方法 → 结果 → 你的批判 |
| **Grant Writing** | Specific Aims + Significance + Innovation + Approach |
| **CV vs Resume** | CV 是学术完整列表（所有论文/演讲/教学），Resume 是精简版 |

## ✏️ 自编模拟题

### 概念题

**Q1**：你在 KINE 601 做第一次学生报告。听众包括 Buchanan、Wright、Lei、Du。你选什么题目最合适？为什么？

**Q2**：描述 IRB、IACUC、COI 的区别，以及分别适用于什么场景。

**Q3**：有人问你能不能把他们的名字加到你的论文上（他们没有做任何实验工作）。从 Authorship 伦理角度，你怎么回应？

### 情景应用题

**Q4**：Buchanan 在 Week 5 介绍了他关于协调动力学的 HKB 模型研究。你课前准备了一个问题："您在 2018 年的 paper 里用了相对相位分析，这和传统的 joint angle 分析相比有什么优势？"——这个问题好在哪？请分析一个好的学术提问的要素。

**Q5**：你需要在系内做一个 15 分钟的 presentation。列出你的 slides 结构和每页时间分配。

## 🔥 高分策略

**这门课通常给 P/F**，但你的表现影响全系对你的第一印象。以下是"表现优秀"的策略：

1. **每场 Seminar 都提问** — 课前 15 分钟 Google 演讲教授的最近论文 → 准备 1 个问题
2. **你的学生报告 = 你在全系面前的名片** — 选 rotation lab 的 preliminary data 最好
3. **记笔记不记内容而记"模板"** — 记教授的 presentation style、slide 设计、Q&A 方式
4. **不要问"您招学生吗"** — 这太浅了，而且你已经有导师了
